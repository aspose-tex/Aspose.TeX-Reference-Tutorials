## 如何在 Java 中將 LaTeX 轉換為 PDF？

`TeXDocument` 類別代表 LaTeX 原始檔，提供載入、處理與渲染文件的方法。  
`save` 將處理後的文件寫入指定的輸出格式，例如 PDF。

使用 `TeXDocument` 類別載入 `.tex` 原始檔，然後呼叫 `save("output.pdf", SaveFormat.Pdf)`——這兩行程式碼即可完成 **how to convert latex** 的全部操作。函式庫會自動解析套件、字型與圖像，產生與原始 LaTeX 版面相同的 PDF。對於大型文件，啟用串流模式即可在檔案超過 500 MB 時仍將記憶體使用量控制在 100 MB 以下。
