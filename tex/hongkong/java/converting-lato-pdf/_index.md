## 如何在 Java 中產生 PDF（從 LaTeX）？

當您詢問 **how to convert LaTeX** 成為 PDF 時，答案可歸納為三個簡單步驟：

1. **Prepare the LaTeX source** – 可以是記憶體中的字串，或是磁碟上的 `.tex` 檔案。  
2. **Create an instance of `TeXDocument`** 從 Aspose.TeX API 建立實例，並載入來源。  
3. **Call the `save` method**，指定 `SaveFormat.PDF` 以產生最終的 PDF 檔案。  

> **Pro tip:** 請確保 LaTeX 原始碼使用 UTF‑8 編碼，以避免字元集問題，特別是處理非 ASCII 符號時。
