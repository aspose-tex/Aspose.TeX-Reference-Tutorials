## 常見問題

**Q:** 我可以在 Web 應用程式中使用 Aspose.TeX 產生 PDF 從 LaTeX 嗎？  
**A:** 可以。此函式庫可在任何 Java 環境中運行，包括 servlet 容器與 Spring Boot 應用程式。

**Q:** 我需要在伺服器上安裝 TeX 發行版嗎？  
**A:** 不需要。Aspose.TeX 內建自己的 TeX 引擎，無需外部相依性。

**Q:** 我該如何處理未隨附的自訂 LaTeX 套件？  
**A:** 您可以將套件檔案放置於與原始檔相同的目錄，或透過 `CompilationOptions` 指定自訂套件路徑。

**Q:** 是否可以直接將產生的 PDF 串流傳送給客戶端，而不儲存至磁碟？  
**A:** 當然可以。使用 `save(OutputStream)` 方法將 PDF 寫入 HTTP 回應串流。

**Q:** 支援哪些 Java 版本？  
**A:** Aspose.TeX 支援 Java 8 及以上版本，包括 Java 11、17 以及更新的 LTS 版本。
