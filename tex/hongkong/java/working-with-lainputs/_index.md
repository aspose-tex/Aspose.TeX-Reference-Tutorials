## 什麼是「從 LaTeX 產生 PDF」？

- **無外部相依性** – 函式庫內建自己的 TeX 引擎。

## 常見問答

**問：我可以在 Web 應用程式中使用 Aspose.TeX 產生 PDF 從 LaTeX 嗎？**  
答：可以。此函式庫可在任何 Java 環境中運作，包括 servlet 容器與 Spring Boot 應用程式。

**問：我需要在伺服器上安裝 TeX 發行版嗎？**  
答：不需要。Aspose.TeX 內建自己的 TeX 引擎，無外部相依性。

**問：如果自訂 LaTeX 套件未隨函式庫一起提供，我該如何處理？**  
答：您可以將套件檔案放在與原始碼相同的目錄，或透過 `CompilationOptions` 指定自訂套件路徑。

**問：能否直接將產生的 PDF 串流傳送給客戶端，而不儲存至磁碟？**  
答：完全可以。使用 `save(OutputStream)` 方法將 PDF 寫入 HTTP 回應串流。

**問：支援哪些 Java 版本？**  
答：Aspose.TeX 支援 Java 8 以及更高版本，包括 Java 11、17 及更新的 LTS 版本。
