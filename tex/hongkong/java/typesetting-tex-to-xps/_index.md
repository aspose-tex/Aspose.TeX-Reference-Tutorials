## 什麼是逐步轉換？

逐步轉換是指將整體轉換流程拆分為清晰且易於管理的階段。透過逐一處理每個階段——初始化、輸入處理、轉換與輸出——你可以完全掌控流程、降低錯誤，並能輕鬆將工作流程適應不同環境。

## 為什麼在 Java 中使用外部串流？

使用外部串流可將 XPS 輸出直接寫入記憶體緩衝區、檔案系統或網路 socket，而無需產生中間檔案。此方式可提升效能、減少 I/O 開銷，並更容易將轉換整合至 Web 服務或雲端管線中。
