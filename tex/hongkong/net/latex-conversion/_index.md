## 常見問題

**Q: 我該如何使用 Aspose.TeX **convert latex to pdf**？**  
A: 建立 `TeXDocument` 實例，載入您的 `.tex` 原始檔，然後呼叫 `Save("output.pdf")`。同一套 API 也可呼叫 `Save("output.png")` 或 `Save("output.svg")` 以輸出其他格式。

**Q: 我可以使用自訂解析度 **export latex as png** 嗎？**  
A: 可以。使用 `PngSaveOptions` 物件在儲存前指定 DPI、背景顏色與影像品質。

**Q: 在 Web 服務中，將 LaTeX **generate pdf from latex** 的最佳方式是什麼？**  
A: 將轉換程式碼部署於無狀態的 .NET Core API 中，盡可能快取產生的 PDF，並將檔案串流回客戶端。

**Q: 轉換的 LaTeX 原始檔大小是否有限制？**  
A: Aspose.TeX 能處理大型文件，但對於極巨檔案，建議提升記憶體限制或將文件分段處理。

**Q: Aspose.TeX 是否支援 **convert latex to svg** 以產生向量圖形？**  
A: 當然支援。使用 `Save("output.svg")` 或 `SvgSaveOptions` 類別來微調輸出。
