## 常見問題

**Q: 如何在 .NET Core 應用程式中將 LaTeX 轉換為 PDF？**  
A: 安裝 Aspose.TeX NuGet 套件，將您的 `.tex` 檔載入 `Document` 物件，然後呼叫 `Save("output.pdf")`。函式庫會在內部處理所有 TeX 轉換。

**Q: 我可以自訂 PDF 輸出（字型、邊距等）嗎？**  
A: 可以。Aspose.TeX 提供如 `DocumentOptions`、`FontSettings`、`PageSetup` 等渲染選項，讓您微調最終的 PDF。

**Q: 是否可以直接將 LaTeX 轉換為 SVG 用於網頁？**  
A: 當然可以。使用 `Save("output.svg")` 方法或 `SvgSaveOptions` 類別，即可從 LaTeX 原始碼產生可縮放向量圖形。

**Q: 需要在伺服器上安裝 LaTeX 發行版嗎？**  
A: 不需要。Aspose.TeX 為純 .NET 函式庫，無需依賴外部 TeX 引擎，讓部署變得簡單且輕量。

**Q: 若需使用自訂 TeX 格式或套件該怎麼辦？**  
A: 您可以在渲染前透過 `TeXEngine` 設定，註冊自訂格式檔或額外套件。
