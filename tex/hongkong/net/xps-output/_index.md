## 快速解答
- **What does Aspose.TeX do?** 它會解析 TeX 標記，並產生 PDF、XPS 或影像輸出。  
- **How to convert TeX to XPS?** 使用 `TeXEngine` 類別，載入你的 .tex 檔案，然後呼叫 `Save(..., SaveFormat.Xps)`。

### 什麼是 XPS？為何要從 TeX 產生它？
XPS（XML Paper Specification）是 Microsoft 的開放、固定版面文件格式。將 TeX 轉換為 XPS 在需要裝置無關、可直接列印的檔案，且能順利整合至 Windows 工作流程、電子閱讀器或歸檔系統時，非常有用。

- **High‑fidelity output** – 完全保留字型、公式與版面配置，與原始檔完全相同。

## 常見問題

**Q: 我可以在不耗盡記憶體的情況下將大型 TeX 專案轉換為 XPS 嗎？**  
A: 可以。使用 `TeXEngine` 的串流選項，並及時釋放中間物件，以降低記憶體使用量。

**Q: 此函式庫支援 TeX 原始檔中嵌入的自訂字型嗎？**  
A: 絕對支援。Aspose.TeX 會遵循 `\usepackage{fontspec}`，並將指定的字型嵌入產生的 XPS 檔案中。

**Q: 我該如何處理 TeX 原始檔的編譯錯誤？**  
A: 捕捉引擎拋出的 `TeXException`；它會提供詳細的行號資訊，協助你修正原始檔。

**Q: 是否能在一次處理中同時產生 PDF 與 XPS？**  
A: 可以。文件渲染完成後，分別以 `SaveFormat.Pdf` 與 `SaveFormat.Xps` 呼叫兩次 `Save`。
