## 常見問題

**Q: 我可以在同一個 PNG 中渲染多個方程式嗎？**  
A: 可以。將 LaTeX 字串串接起來，或使用 `Document` 類別在渲染前排版多個公式。

**Q: Aspose.TeX 支援其他影像格式嗎？**  
A: 當然支援。除了 PNG，您也可透過變更 `ImageFormat` 屬性輸出 JPEG、BMP、GIF 與 TIFF。

**Q: 如何提升批次處理的渲染速度？**  
A: 重複使用同一個 `Renderer` 實例，並預先載入所需字型；可減少每次轉換的額外開銷。

**Q: 能否將 LaTeX 渲染成透明背景的 PNG？**  
A: 在呼叫 `Save` 前，於渲染選項中將背景顏色設為 `Color.Transparent`。

**Q: 需要哪個版本的 .NET？**  
A: Aspose.TeX 支援 .NET Framework 4.5 以上、.NET Core 3.1 以上、.NET 5 與 .NET 6。
