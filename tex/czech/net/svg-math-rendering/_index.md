## Co je „generovat SVG z LaTeXu“?

Generování SVG z LaTeXu znamená převod matematického výrazu formátovaného v LaTeXu do souboru Scalable Vector Graphics (SVG). SVG je nezávislé na rozlišení, lehké a ideální pro webové nebo desktopové vykreslování, což z něj činí perfektní prostředek pro zobrazování složitých vzorců s pixelovou dokonalou jasností.

## Často kladené otázky

**Q: Mohu použít vygenerované SVG soubory na webu bez další konverze?**  
A: Ano—SVG je nativně podporováno všemi moderními prohlížeči, takže můžete výstup vložit přímo do HTML nebo CSS.

**Q: Jak změním výchozí písmo pro vykreslenou matematiku?**  
A: Použijte vlastnost `FontFamily` konfigurace `TeXRenderer` k určení libovolného nainstalovaného TrueType/OpenType písma.

**Q: Je možné vykreslovat LaTeX rovnice, které obsahují barvu nebo vlastní makra?**  
A: Rozhodně. Aspose.TeX zpracovává standardní LaTeX balíčky pro barvy a umožňuje definovat makra pomocí metody `AddMacro`.

**Q: Jaká bude velikost vygenerovaného SVG?**  
A: Rozměry SVG jsou automaticky vypočítány na základě ohraničujícího rámečku rovnice, ale můžete je přepsat pomocí nastavení `Width` a `Height`.

**Q: Podporuje knihovna hromadné zpracování více rovnic?**  
A: Ano—můžete projít kolekci LaTeX řetězců a vykreslit každý do samostatného SVG souboru s minimální režijní zátěží.
