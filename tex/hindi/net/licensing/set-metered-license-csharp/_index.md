# Aspose.TeX (C#) के लिए लाइसेंस कैसे सेट करें

## परिचय

यदि आप C# एप्लिकेशन में TeX फ़ाइलों के साथ काम करने की योजना बना रहे हैं, तो सबसे पहला कदम **Aspose.TeX के लिए लाइसेंस कैसे सेट करें** यह जानना है। मीटरड लाइसेंस सेट करने से न केवल रन‑टाइम प्रतिबंध हटते हैं बल्कि लाइब्रेरी की पूरी फ़ीचर सेट तक पहुँच मिलती है। इस गाइड में हम पूरी प्रक्रिया—SDK डाउनलोड करने से लेकर लाइसेंस सक्रिय होने की पुष्टि तक—परिचित कराएँगे, ताकि आप तुरंत TeX‑सक्षम समाधान बनाना शुरू कर सकें।

## त्वरित उत्तर
- **मीटरड लाइसेंस क्या है?** एक हल्का लाइसेंस मॉडल जो सार्वजनिक/निजी कुंजियों के माध्यम से उपयोग को वैध करता है, बिना स्थानीय लाइसेंस फ़ाइल की आवश्यकता के।  
- **क्या विकास के लिए लाइसेंस चाहिए?** हाँ, सभी फ़ीचर अनलॉक करने के लिए विकास और उत्पादन दोनों में मीटरड लाइसेंस आवश्यक है।  
- **कौन से .NET संस्करण समर्थित हैं?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7।  
- **क्या बाद में कुंजियों को बदल सकता हूँ?** बिल्कुल—नयी कुंजियों के साथ `SetMeteredKey` को फिर से कॉल करें।  
- **लाइसेंस काम कर रहा है, यह कैसे पुष्टि करें?** `Aspose.TeX.Metered.IsMetered()` का उपयोग करके true/false परिणाम प्राप्त करें।

## मीटरड लाइसेंस क्या है?

Aspose.TeX के लिए मीटरड लाइसेंस प्रत्येक बार जब आपका एप्लिकेशन शुरू होता है, Aspose के लाइसेंसिंग सर्वर को एक हल्का अनुरोध भेजता है। सर्वर आपके द्वारा प्रदान की गई सार्वजनिक और निजी कुंजियों को वैध करता है और एक उपयोग टोकन लौटाता है। यह तरीका भौतिक लाइसेंस फ़ाइल को शिप करने की आवश्यकता को समाप्त करता है और आवश्यकता पड़ने पर कुंजियों को आसानी से बदलने की सुविधा देता है।

## Aspose.TeX के लिए मीटरड लाइसेंस क्यों उपयोग करें?

- **फ़ाइल डिप्लॉयमेंट नहीं** – कुंजियाँ सीधे कोड में एम्बेड होती हैं।  
- **आसान कुंजी रोटेशन** – लाइसेंस फ़ाइल को पुनः डिप्लॉय किए बिना कुंजियों को अपडेट करें।  
- **सटीक उपयोग ट्रैकिंग** – Aspose प्रत्येक सक्रियता को रिकॉर्ड करता है, जिससे आप अपनी सब्सक्रिप्शन सीमा के भीतर रह सकें।  
- **पूरा फ़ीचर एक्सेस** – लाइसेंस वैध होने पर सभी API क्षमताएँ अनलॉक हो जाती हैं।

## नेमस्पेस इम्पोर्ट करें

## C# Aspose लाइसेंस सेट – चरण‑दर‑चरण गाइड

### चरण 1: मीटरड लाइसेंस सेट करें (लाइसेंस कैसे सेट करें)

पहला कोड स्निपेट दिखाता है कि मीटरड कुंजियों का उपयोग करके **लाइसेंस कैसे सेट करें**। इसे अपने एप्लिकेशन स्टार्टअप रूटीन (जैसे `Main` या `Startup.cs`) में शुरुआती स्थान पर रखें।

```csharp
// ExStart:SetMeteredLicense
// Set metered public and private keys.
new Aspose.TeX.Metered().SetMeteredKey(
    "<type public key here>",
    "<type private key here>");
// ExEnd:SetMeteredLicense
```

`<type public key here>` और `<type private key here>` को Aspose से प्राप्त कुंजियों से बदलें।

### चरण 2: प्रोजेक्ट में इंटीग्रेट करें

लाइसेंस सेट होने के बाद, आप किसी भी Aspose.TeX क्लास का स्वतंत्र रूप से उपयोग कर सकते हैं—LaTeX कंपाइल करना, PDF में बदलना, इमेज निकालना आदि। अतिरिक्त लाइसेंस कोड की आवश्यकता नहीं है।

### चरण 3: लाइसेंस की पुष्टि करें

यह एक अच्छी प्रैक्टिस है कि लाइसेंस सफलतापूर्वक लागू हुआ है, यह पुष्टि करें। निम्नलिखित स्निपेट कंसोल पर स्पष्ट संदेश प्रिंट करता है।

```csharp
// ExStart:VerifyMeteredLicense
if (Aspose.TeX.Metered.IsMetered())
{
    Console.WriteLine("Metered license is set successfully!");
}
else
{
    Console.WriteLine("Metered license is not set!");
}
// ExEnd:VerifyMeteredLicense
```

यदि आप “Metered license is set successfully!” देखते हैं, तो आप तैयार हैं।
