# .NET में Aspose.TeX के साथ LaTeX को SVG में बदलें

## परिचय

जब आपको वेब पेज, PDF या डेस्कटॉप ऐप्स के लिए स्पष्ट, रिज़ॉल्यूशन‑इंडिपेंडेंट गणित ग्राफ़िक्स चाहिए होते हैं, तो LaTeX को SVG में बदलना एक सामान्य आवश्यकता होती है। .NET दुनिया में, **Aspose.TeX** एक समर्पित API प्रदान करता है जो आपको केवल कुछ लाइनों के कोड में **convert LaTeX to SVG** करने देता है, साथ ही स्टाइलिंग, स्केलिंग और रंग पर पूर्ण नियंत्रण देता है। यह ट्यूटोरियल आपको पूरी पाइपलाइन के माध्यम से ले जाता है—रेंडरिंग विकल्प सेट करने से लेकर अंतिम SVG प्रदर्शित करने तक—ताकि आप किसी भी .NET प्रोजेक्ट में उच्च‑गुणवत्ता वाले समीकरण एकीकृत कर सकें।

## “convert LaTeX to SVG” क्या है?

`convert LaTeX to SVG` वह प्रक्रिया है जिसमें LaTeX गणित अभिव्यक्ति को एक स्केलेबल वेक्टर ग्राफ़िक में बदल दिया जाता है जो किसी भी आकार पर पूरी स्पष्टता बनाए रखती है। अपनी LaTeX स्ट्रिंग लोड करें, Aspose.TeX को कंपाइल करने दें, और लाइब्रेरी एक SVG फ़ाइल उत्पन्न करती है जिसे आप कहीं भी पिक्सेलेशन के बिना एम्बेड कर सकते हैं। यह सीधा‑उत्तर पैराग्राफ आपको ठीक‑ठीक बताता है कि क्या होता है, और ऊपर दिया गया परिभाषा एंकर AI एक्सट्रैक्शन नियमों को पूरा करता है।

## नेमस्पेस इम्पोर्ट करें

`Aspose.TeX` नेमस्पेस उन सभी क्लासेज़ को प्रदान करता है जिनकी आपको समीकरण रेंडर करने की आवश्यकता है। इसे अपनी फ़ाइल के शीर्ष पर इम्पोर्ट करें:

```csharp
using Aspose.TeX;
```

अब हम चरण‑दर‑चरण रेंडरिंग पाइपलाइन को देखते हैं।

## चरण 1: रेंडरिंग विकल्प बनाएं

MathRendererOptions बेस क्लास है जो LaTeX को विभिन्न फॉर्मैट में रेंडर करने के लिए सेटिंग्स रखती है। SvgMathRendererOptions इससे व्युत्पन्न है और SVG‑विशिष्ट प्रॉपर्टीज़ जोड़ती है।

```csharp
using Aspose.TeX.Features;
```

## चरण 2: प्रीऐम्बल निर्दिष्ट करें

Preamble प्रॉपर्टी आपको मुख्य समीकरण से पहले प्रोसेस होने वाले LaTeX पैकेज और कमांड जोड़ने की अनुमति देती है।

```csharp
// Create rendering options.
MathRendererOptions options = new SvgMathRendererOptions();
```

## चरण 3: स्केलिंग फैक्टर और रंग सेट करें

options.Scale आउटपुट SVG का आकार नियंत्रित करता है, जबकि options.TextColor और options.BackgroundColor क्रमशः फ़ोरग्राउंड और बैकग्राउंड रंग निर्धारित करते हैं।

```csharp
// Specify the preamble.
options.Preamble = @"\usepackage{amsmath}
\usepackage{amsfonts}
\usepackage{amssymb}
\usepackage{color}";
```

## चरण 4: आउटपुट विकल्प कॉन्फ़िगर करें

OutputFile वह पाथ निर्दिष्ट करता है जहाँ उत्पन्न SVG सहेजा जाएगा, और options.EmbedFonts निर्धारित करता है कि फ़ॉन्ट्स SVG में एम्बेड हों या नहीं।

```csharp
// Specify the scaling factor (e.g., 300%).
options.Scale = 3000;

// Specify the foreground color.
options.TextColor = System.Drawing.Color.Black;

// Specify the background color.
options.BackgroundColor = System.Drawing.Color.White;
```

## चरण 5: LaTeX गणित समीकरण रेंडर करें

MathRenderer वह इंजन है जो LaTeX स्ट्रिंग और रेंडरिंग विकल्पों को लेता है और अंतिम SVG दस्तावेज़ बनाता है।

```csharp
// Specify the output stream for the log file.
options.LogStream = new System.IO.MemoryStream();

// Specify whether to show the terminal output on the console or not.
options.ShowTerminal = true;
```

## चरण 6: परिणाम प्रदर्शित करें

SvgDocument उत्पन्न SVG को दर्शाता है और इसे डिस्क पर सहेजा जा सकता है या सीधे वेब रिस्पॉन्स में स्ट्रीम किया जा सकता है।

```csharp
// Create the output stream for the formula image.
using (System.IO.Stream stream = System.IO.File.Open(
    System.IO.Path.Combine("Your Output Directory", "math-formula.svg"), System.IO.FileMode.Create))
{
    // Run rendering.
    new SvgMathRenderer().Render(@"\begin{equation*}
    e^x = x^{\color{red}0} + x^{\color{red}1} + \frac{x^{\color{red}2}}{2} + \frac{x^{\color{red}3}}{6} + \cdots = \sum_{n\geq 0} \frac{x^{\color{red}n}}{n!}
\end{equation*}", stream, options, out size);
}
```
