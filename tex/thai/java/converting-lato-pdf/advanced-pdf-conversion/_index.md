## นำเข้าแพ็กเกจ

เพื่อเริ่มต้น เรามานำเข้าแพ็กเกจที่จำเป็นสำหรับโปรเจ็กต์ Java ของเรากัน:

```java
package com.aspose.tex.LaTeXPdfConversionAlternative;

import java.io.FileOutputStream;
import java.io.IOException;
import java.io.OutputStream;

import com.aspose.tex.OutputFileSystemDirectory;
import com.aspose.tex.TeXConfig;
import com.aspose.tex.TeXJob;
import com.aspose.tex.TeXOptions;
import com.aspose.tex.rendering.PdfDevice;
import com.aspose.tex.rendering.PdfSaveOptions;
```

ต่อไปนี้คือขั้นตอนโดยละเอียดของโค้ดตัวอย่าง:

## การสร้าง PDF จาก LaTeX ด้วย Java – ภาพรวม

### ขั้นตอนที่ 1: ตั้งค่าใบอนุญาต

```java
Utils.setLicense();
```

ตรวจสอบให้แน่ใจว่าได้ตั้งค่าใบอนุญาต Aspose.TeX ของคุณโดยใช้วิธีการที่ให้ไว้ หากไม่มีใบอนุญาต ไฟล์ PDF ที่ได้จะมีลายน้ำ

### ขั้นตอนที่ 2: กำหนดค่าตัวเลือกการแปลง

```java
TeXOptions options = TeXOptions.consoleAppOptions(TeXConfig.objectLaTeX());
```

สร้างตัวเลือก TeX สำหรับรูปแบบ Object LaTeX โดยใช้ส่วนขยายเอ็นจิ้น Object TeX การกำหนดค่านี้เหมาะอย่างยิ่งเมื่อคุณต้องการให้เอ็นจิ้นแก้ไขทรัพยากรภายนอกทั้งหมดโดยอัตโนมัติ

### ขั้นตอนที่ 3: ระบุไดเร็กทอรีเอาต์พุต

```java
options.setOutputWorkingDirectory(new OutputFileSystemDirectory("Your Output Directory"));
```

ตั้งค่าไดเร็กทอรีการทำงานของระบบไฟล์สำหรับไฟล์ PDF ที่ได้ แทนที่ `"Your Output Directory"` ด้วยเส้นทางสัมบูรณ์หรือสัมพัทธ์ที่ควรเขียนไฟล์ PDF

### ขั้นตอนที่ 4: เริ่มต้นตัวเลือกการบันทึก PDF

```java
options.setSaveOptions(new PdfSaveOptions());
```

เริ่มต้นตัวเลือกสำหรับการบันทึกในรูปแบบ PDF `PdfSaveOptions` ช่วยให้คุณปรับแต่งการบีบอัด ฝังฟอนต์ และควบคุมการกำหนดเวอร์ชัน PDF ได้

### ขั้นตอนที่ 5: เรียกใช้การแปลง

```java
new TeXJob("Your Input Directory" + "hello-world.ltx", new PdfDevice(pdfStream), options).run();
```

เรียกใช้การแปลง **java generate pdf from latex** โดยใช้ Aspose.TeX `PdfDevice` จะส่งไฟล์ PDF ที่สร้างขึ้นไปยัง `pdfStream` ที่คุณระบุไว้

### ขั้นตอนที่ 6: ปิดสตรีมเอาต์พุต

```java
if (pdfStream != null)
    pdfStream.close();
```

ปิดสตรีมเอาต์พุตเพื่อให้แน่ใจว่ามีการจัดการทรัพยากรอย่างเหมาะสมและหลีกเลี่ยงการล็อกไฟล์

## ปัญหาและวิธีแก้ไขทั่วไป

| ปัญหา | สาเหตุ | วิธีแก้ไข |

|-------|-------|----------|
| **ฟอนต์หายไป** | เอ็นจิ้น LaTeX ไม่สามารถค้นหาฟอนต์ที่จำเป็นได้ | ติดตั้งฟอนต์ที่หายไปบนระบบปฏิบัติการโฮสต์หรือฝังฟอนต์ผ่าน `PdfSaveOptions.setEmbedAllFonts(true)` |
| **ลายน้ำปรากฏ** | ไม่ได้ตั้งค่าใบอนุญาตหรือใบอนุญาตหมดอายุ | ตรวจสอบว่า `Utils.setLicense()` ชี้ไปยังไฟล์ `.lic` ที่ถูกต้องและใบอนุญาตยังไม่หมดอายุ |
| **ไม่ได้สร้างไดเร็กทอรีเอาต์พุต** | เส้นทางไม่มีอยู่และ `OutputFileSystemDirectory` ไม่สามารถสร้างได้ | ตรวจสอบให้แน่ใจว่าไดเร็กทอรีมีอยู่หรือให้สิทธิ์การเขียนแก่กระบวนการเพื่อสร้างไดเร็กทอรี |
| **เอกสารขนาดใหญ่ทำให้เกิดข้อผิดพลาด OutOfMemoryError** | PDF ถูกสร้างขึ้นในหน่วยความจำทั้งหมด | ใช้ API สตรีมมิ่ง (`PdfDevice` กับ `FileOutputStream`) เพื่อเขียนลงดิสก์โดยตรง |
