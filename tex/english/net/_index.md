## Frequently Asked Questions

**Q: How do I convert LaTeX to PDF in a .NET Core application?**  
A: Install the Aspose.TeX NuGet package, load your `.tex` file into a `Document` object, and call `Save("output.pdf")`. The library handles all TeX processing internally.

**Q: Can I customize the PDF output (fonts, margins, etc.)?**  
A: Yes. Aspose.TeX exposes rendering options such as `DocumentOptions`, `FontSettings`, and `PageSetup` that let you fine‑tune the final PDF.

**Q: Is it possible to convert LaTeX directly to SVG for web use?**  
A: Absolutely. Use the `Save("output.svg")` method or the `SvgSaveOptions` class to generate scalable vector graphics from your LaTeX source.

**Q: Do I need to install a LaTeX distribution on the server?**  
A: No. Aspose.TeX is a pure .NET library and does not rely on external TeX engines, making deployment simple and lightweight.

**Q: What if I need to use a custom TeX format or package?**  
A: You can register custom format files or additional packages via the `TeXEngine` configuration before rendering.
