## Frequently Asked Questions

**Q: How do I generate PDF from TeX without installing a local LaTeX distribution?**  
A: Aspose.TeX embeds the TeX engine, so you can call the API directly from your .NET code and obtain a PDF instantly.

**Q: How to convert LaTeX PDF to other formats like SVG?**  
A: Use the same Aspose.TeX API with the appropriate `Save` method (e.g., `Save("output.svg", SaveOptions.CreateSvgSaveOptions())`). This is covered in the “latex conversion to svg” tutorials.

**Q: Can I render LaTeX equations as PNG for web thumbnails?**  
A: Yes – the “render latex png” guide shows how to call `Save("eq.png", SaveOptions.CreatePngSaveOptions())` to get high‑resolution PNG images.

**Q: Do I need a separate license for each output format?**  
A: No. A single Aspose.TeX license covers all supported output formats (PDF, PNG, SVG, XPS, etc.).

**Q: What are the common pitfalls when generating PDFs from TeX?**  
A: Missing fonts, unsupported packages, or incorrect file paths. The “how to convert latex pdf” tutorial includes troubleshooting tips for these issues.
