## Perguntas Frequentes

**Q: Posso renderizar várias equações em um único PNG?**  
A: Sim. Concatene strings LaTeX ou use a classe `Document` para organizar várias fórmulas antes da renderização.

**Q: O Aspose.TeX suporta outros formatos de imagem?**  
A: Absolutamente. Além de PNG, você pode gerar JPEG, BMP, GIF e TIFF alterando a propriedade `ImageFormat`.

**Q: Como melhorar a velocidade de renderização para processamento em lote?**  
A: Reutilize a mesma instância de `Renderer` e pré‑carregue as fontes necessárias; isso reduz a sobrecarga em cada conversão.

**Q: É possível renderizar LaTeX para um PNG transparente?**  
A: Defina a cor de fundo como `Color.Transparent` nas opções de renderização antes de chamar `Save`.

**Q: Qual versão do .NET é necessária?**  
A: Aspose.TeX suporta .NET Framework 4.5+, .NET Core 3.1+, .NET 5 e .NET 6.
