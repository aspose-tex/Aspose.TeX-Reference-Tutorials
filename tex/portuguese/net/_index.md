## Perguntas Frequentes

**Q: Como converto LaTeX para PDF em uma aplicação .NET Core?**  
**A:** Instale o pacote NuGet Aspose.TeX, carregue seu arquivo `.tex` em um objeto `Document` e chame `Save("output.pdf")`. A biblioteca lida com todo o processamento TeX internamente.

**Q: Posso personalizar a saída PDF (fontes, margens, etc.)?**  
**A:** Sim. Aspose.TeX expõe opções de renderização como `DocumentOptions`, `FontSettings` e `PageSetup` que permitem ajustar finamente o PDF final.

**Q: É possível converter LaTeX diretamente para SVG para uso web?**  
**A:** Absolutamente. Use o método `Save("output.svg")` ou a classe `SvgSaveOptions` para gerar gráficos vetoriais escaláveis a partir do seu código-fonte LaTeX.

**Q: Preciso instalar uma distribuição LaTeX no servidor?**  
**A:** Não. Aspose.TeX é uma biblioteca .NET pura e não depende de engines TeX externas, tornando a implantação simples e leve.

**Q: E se eu precisar usar um formato ou pacote TeX personalizado?**  
**A:** Você pode registrar arquivos de formato personalizados ou pacotes adicionais via a configuração `TeXEngine` antes da renderização.
