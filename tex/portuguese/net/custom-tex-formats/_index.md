## Perguntas Frequentes

**Q: Posso usar formatos TeX personalizados em uma aplicação comercial?**  
A: Sim. Com uma licença válida do Aspose.TeX você pode implantar formatos personalizados em qualquer projeto comercial.

**Q: Preciso instalar uma distribuição completa do TeX?**  
A: Não. Aspose.TeX lida com todo o processamento TeX internamente, portanto você só precisa da biblioteca .NET.
