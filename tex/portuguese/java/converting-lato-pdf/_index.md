## Como gerar PDF a partir de LaTeX em Java?

Quando você pergunta **como converter LaTeX** em PDF, a resposta se resume a três passos simples:

1. **Prepare a fonte LaTeX** – seja como uma string na memória ou como um arquivo `.tex` no disco.  
2. **Crie uma instância de `TeXDocument`** a partir da API Aspose.TeX e carregue a fonte.  
3. **Chame o método `save`** especificando `SaveFormat.PDF` para gerar o arquivo PDF final.

> **Dica profissional:** Mantenha sua fonte LaTeX codificada em UTF‑8 para evitar problemas de conjunto de caracteres, especialmente ao lidar com símbolos não‑ASCII.
