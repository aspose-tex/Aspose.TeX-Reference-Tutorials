## 常见问题解答

**问：我可以在 Web 应用程序中使用 Aspose.TeX 从 LaTeX 生成 PDF 吗？**  
答：可以。该库可在任何 Java 环境中运行，包括 servlet 容器和 Spring Boot 应用程序。

**问：我需要在服务器上安装 TeX 发行版吗？**  
答：不需要。Aspose.TeX 包含其自带的 TeX 引擎，无需外部依赖。

**问：如何处理未打包的自定义 LaTeX 包？**  
答：您可以将包文件放置在与源文件相同的目录中，或通过 `CompilationOptions` 指定自定义包路径。

**问：是否可以直接将生成的 PDF 流式传输给客户端，而不保存到磁盘？**  
答：完全可以。使用 `save(OutputStream)` 方法将 PDF 写入 HTTP 响应流。

**问：支持哪些 Java 版本？**  
答：Aspose.TeX 支持 Java 8 及更高版本，包括 Java 11、17 以及更新的 LTS 发行版。
