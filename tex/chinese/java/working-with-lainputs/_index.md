## 常见问题

**Q: 我可以在 Web 应用程序中使用 Aspose.TeX 从 LaTeX 生成 PDF 吗？**  
A: 可以。该库可在任何 Java 环境中运行，包括 servlet 容器和 Spring Boot 应用。

**Q: 服务器上需要安装 TeX 发行版吗？**  
A: 不需要。Aspose.TeX 自带 TeX 引擎，无外部依赖。

**Q: 如何处理未随库打包的自定义 LaTeX 宏包？**  
A: 可以将宏包文件放在与源文件相同的目录，或通过 `CompilationOptions` 指定自定义宏包路径。

**Q: 能否直接将生成的 PDF 流式传输给客户端，而不保存到磁盘？**  
A: 完全可以。使用 `save(OutputStream)` 方法将 PDF 写入 HTTP 响应流。

**Q: 支持哪些 Java 版本？**  
A: Aspose.TeX 支持 Java 8 及更高版本，包括 Java 11、17 以及更新的 LTS 版本。
