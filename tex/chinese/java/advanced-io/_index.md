## 常见问题解答

**Q: 我可以生成矢量图像（SVG）格式吗？**  
A: 是的，在渲染选项中将输出格式设为 `Svg` 即可获得可缩放的矢量图形。

**Q: 如何处理包含外部宏包的 TeX 文件？**  
A: 将所需的 `.sty` 文件放在同一输入目录中，或将其路径添加到 `TeXProcessor` 的 `PackageSearchPath`。

**Q: 是否可以直接X 而无需写入磁盘？**  
A: 完全可以——Aspose.TeX 完全支持基于流的输入，非常适合 Web 服务和微服务。

**Q: Aspose.TeX 使用何种授权模式？**  
A: 提供永久授权，可选支持续订；也提供免费评估授权。

**Q: 该库是否支持 TeX 源中的 Unicode 字符？**  
A: 是的，Aspose.TeX 开箱即支持 UTF‑8 编码的 TeX 文件。

**Q: 我还能使用同一库 **convert latex to pdf** 吗？**  
A `Pdf` 即可直接将 LaTeX 文档转换为 PDF。

**Q: 如何 **render latex as svg** 以实现网页友好的缩放？**  
A: 使用 `Svg` 输出格式，并可选地调整 `DpiX`/`DpiY` 设置以获得更细致的细节。

**Q: 对于高分辨率需求，最佳的 **generate png from latex** 方法是什么？**  
A: 在调用 `process` 之前，提升 `RenderOptions` 中的 `Resolution` 属性（例如 300 dpi）。
