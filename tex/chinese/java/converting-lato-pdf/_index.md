## 如何在 Java 中生成 PDF（从 LaTeX）？
当您询问 **how to convert LaTeX** 为 PDF 时，答案归结为以下三步：

1. **准备 LaTeX 源** —— 可以是内存中的字符串，也可以是磁盘上的 `.tex` 文件。  
2. **创建 `TeXDocument` 实例**（来自 Aspose.TeX API）并加载源文件。  
3. **调用 `save` 方法**，指定 `SaveFormat.PDF`，即可生成最终的 PDF 文件。

> **专业提示：** 请确保 LaTeX 源文件使用 UTF‑8 编码，以避免字符集问题，尤其是在处理非 ASCII 符号时。
