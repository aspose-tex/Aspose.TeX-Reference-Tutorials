## 常见问题

**Q: 如何使用 Aspose.TeX **convert latex to pdf**？**  
A: 实例化 `TeXDocument`，加载您的 `.tex` 源文件，然后调用 `Save("output.pdf")`。相同的 API 也可以调用 `Save("output.png")` 或 `Save("output.svg")` 来生成其他格式。

**Q: 我可以使用自定义分辨率 **export latex as png** 吗？**  
A: 可以。使用 `PngSaveOptions` 对象在保存前指定 DPI、背景颜色和图像质量。

**Q: 在 Web 服务中 **generate pdf from latex** 的最佳方式是什么？**  
A: 将转换代码部署在无状态的 .NET Core API 中，尽可能缓存生成的 PDF，并将文件流式返回给客户端。

**Q: 转换的 LaTeX 源文件大小是否有限制？**  
A: Aspose.TeX 能处理大型文档，但对于极大的文件，建议增加内存限制或将文档分段处理。

**Q: Aspose.TeX 是否支持 **convert latex to svg** 用于矢量图形？**  
A: 当然。使用 `Save("output.svg")` 或 `SvgSaveOptions` 类来微调输出。
