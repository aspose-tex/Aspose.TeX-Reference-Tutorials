## 常见问题

**问：我可以在网页上直接使用生成的 SVG 文件而无需额外转换吗？**  
答：是的—SVG 被所有现代浏览器原生支持，您可以直接将输出嵌入 HTML 或 CSS。

**问：如何更改渲染数学的默认字体？**  
答：使用 `TeXRenderer` 配置的 `FontFamily` 属性来指定任何已安装的 TrueType/OpenType 字体。

**问：是否可以渲染包含颜色或自定义宏的 LaTeX 方程？**  
答：当然。Aspose.TeX 处理标准的 LaTeX 颜色宏包，并允许通过 `AddMacro` 方法定义宏。

**问：生成的 SVG 大小是多少？**  
答：SVG 尺寸会根据公式的边界框自动计算，但您可以使用 `Width` 和 `Height` 设置进行覆盖。

**问：该库是否支持批量处理多个方程？**  
答：是的—您可以遍历 LaTeX 字符串集合，将每个渲染为独立的 SVG 文件，开销极小。
