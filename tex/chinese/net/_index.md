## 常见问题

**Q: 如何在 .NET Core 应用程序中将 LaTeX 转换为 PDF？**  
A: 安装 Aspose.TeX NuGet 包，将您的 `.tex` 文件加载到 `Document` 对象中，然后调用 `Save("output.pdf")`。库会在内部处理所有 TeX 处理。

**Q: 我可以自定义 PDF 输出（字体、页边距等）吗？**  
A: 可以。Aspose.TeX 提供了渲染选项，如 `DocumentOptions`、`FontSettings` 和 `PageSetup`，让您可以微调最终的 PDF。

**Q: 能否直接将 LaTeX 转换为 SVG 用于网页？**  
A: 当然可以。使用 `Save("output.svg")` 方法或 `SvgSaveOptions` 类即可从 LaTeX 源生成可缩放矢量图形。

**Q: 服务器上需要安装 LaTeX 发行版吗？**  
A: 不需要。Aspose.TeX 是纯 .NET 库，不依赖外部 TeX 引擎，部署简便且轻量。

**Q: 如果需要使用自定义 TeX 格式或宏包怎么办？**  
A: 您可以在渲染前通过 `TeXEngine` 配置注册自定义格式文件或额外的宏包。
