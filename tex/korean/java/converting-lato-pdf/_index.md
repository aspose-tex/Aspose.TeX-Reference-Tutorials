## latex to pdf java 변환 수행 방법

When you ask **how to convert LaTeX** into a PDF, the answer boils down to three simple steps:

1. **Prepare the LaTeX source** – 메모리 내 문자열이든 디스크의 `.tex` 파일이든.  
2. **Create an instance of `TeXDocument`** – Aspose.TeX API에서 `TeXDocument` 인스턴스를 생성하고 소스를 로드합니다.  
3. **Call the `save` method** – `SaveFormat.PDF`를 지정하여 최종 PDF 파일을 생성합니다.

> **Pro tip:** LaTeX 소스를 UTF‑8 인코딩으로 유지하여 문자 집합 문제를 방지하세요, 특히 비 ASCII 기호를 다룰 때.
