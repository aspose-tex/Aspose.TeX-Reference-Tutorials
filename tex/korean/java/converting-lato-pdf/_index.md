## Java에서 LaTeX로부터 PDF를 생성하는 방법
**LaTeX를 PDF로 변환**하는 방법을 물으신다면, 답은 세 단계로 요약됩니다:

1. **LaTeX 소스 준비** – 메모리 상의 문자열이든 디스크에 있는 `.tex` 파일이든 상관없습니다.  
2. **Aspose.TeX API의 `TeXDocument` 인스턴스 생성** 후 소스를 로드합니다.  
3. **`save` 메서드에 `SaveFormat.PDF` 지정**하여 최종 PDF 파일을 생성합니다.

> **Pro tip:** 문자 인코딩 문제를 방지하려면 LaTeX 소스를 UTF‑8로 인코딩하세요. 특히 비 ASCII 기호를 사용할 때 중요합니다.
