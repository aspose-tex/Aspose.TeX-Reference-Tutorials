### Aspose.TeX란?

Aspose.TeX는 TeX/LaTeX 소스 파일을 구문 분석하고 코드에서 직접 PDF, XPS, PNG, SVG 및 기타 형식을 생성하는 Java 라이브러리입니다. TeX 엔진의 복잡성을 추상화하여 개발자에게 엔터프라이즈 애플리케이션에 과학 출판 워크플로를 삽입할 수 있는 간단한 API를 제공합니다.

## Java에서 LaTeX를 PDF로 변환하는 방법?

`TeXDocument` 클래스는 LaTeX 소스 파일을 나타내며 문서를 로드, 처리 및 렌더링하는 메서드를 제공합니다.  
`save`는 처리된 문서를 PDF와 같은 선택된 출력 형식으로 기록합니다.

`.tex` 소스를 `TeXDocument` 클래스로 로드하고 `save("output.pdf", SaveFormat.Pdf)`를 호출하면 – 이 두 줄의 단일 시퀀스가 전체 **how to convert latex** 작업을 수행합니다. 라이브러리는 패키지, 글꼴 및 이미지를 자동으로 해결하여 원본 LaTeX 레이아웃과 일치하는 PDF를 제공합니다. 큰 문서의 경우 스트리밍 모드를 활성화하여 파일 크기가 500 MB를 초과하더라도 메모리 사용량을 100 MB 이하로 유지할 수 있습니다.
