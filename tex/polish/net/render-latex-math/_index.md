## Najczęściej zadawane pytania

**Q: Czy mogę renderować wiele równań w jednym pliku PNG?**  
A: Tak. Połącz łańcuchy LaTeX lub użyj klasy `Document`, aby rozłożyć kilka formuł przed renderowaniem.

**Q: Czy Aspose.TeX obsługuje inne formaty obrazów?**  
A: Oczywiście. Oprócz PNG możesz wyjść w formatach JPEG, BMP, GIF i TIFF, zmieniając właściwość `ImageFormat`.

**Q: Jak mogę przyspieszyć renderowanie przy przetwarzaniu wsadowym?**  
A: Ponownie używaj tej samej instancji `Renderer` i wstępnie ładowane wymagane czcionki; to zmniejsza narzut przy każdej konwersji.

**Q: Czy można renderować LaTeX do przezroczystego PNG?**  
A: Ustaw kolor tła na `Color.Transparent` w opcjach renderowania przed wywołaniem `Save`.

**Q: Jaką wersję .NET trzeba mieć?**  
A: Aspose.TeX obsługuje .NET Framework 4.5+, .NET Core 3.1+, .NET 5 oraz .NET 6.
