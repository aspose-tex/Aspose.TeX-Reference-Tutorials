# التحويل خطوة بخطوة – Java إنشاء فواتير قابلة للطباعة: LaTeX إلى تنسيق XPS

## مقدمة

إذا كنت بحاجة إلى **java create printable invoices** من مصادر LaTeX داخل تطبيق Java، فقد وصلت إلى الدرس المناسب. باستخدام **Aspose.TeX for Java** يمكنك عرض ملفات LaTeX، ومعالجة الصيغ المعقدة، وإنتاج ملفات XPS عالية الجودة التي تُطبع بالضبط كما تتوقع. خلال الدقائق القليلة القادمة سنستعرض سير العمل بالكامل، نشرح لماذا هذه الطريقة مثالية لإنشاء الفواتير، ونظهر لك كيفية تخصيص التحويل لأنابيب التقارير الخاصة بك. يوضح هذا الدليل **step by step conversion** كامل يمكنك نسخه ولصقه في أي مشروع Java.

## استيراد الحزم

كتلة `import` تجلب الفئات الأساسية إلى النطاق. الحفاظ على هذه القائمة الدقيقة يضمن تجميع الكود دون تعديل.

```java
package com.aspose.tex.LaTeXXpsConversionSimplest;

import java.io.ByteArrayInputStream;
import java.io.IOException;
import java.util.Calendar;
import java.util.GregorianCalendar;

import com.aspose.tex.InputFileSystemDirectory;
import com.aspose.tex.Interaction;
import com.aspose.tex.OutputFileSystemDirectory;
import com.aspose.tex.TeXConfig;
import com.aspose.tex.TeXJob;
import com.aspose.tex.TeXOptions;
import com.aspose.tex.rendering.XpsDevice;
import com.aspose.tex.rendering.XpsSaveOptions;

import util.Utils;
```

الآن دعنا نستعرض خطوات التحويل، موضحين هدف كل جزء من الكود.

## الخطوة 1: تعيين مجلدات الإدخال والإخراج

تحتاج إلى إخبار Aspose.TeX بمكان العثور على ملف المصدر `.ltx` وأين كتابة ملف XPS الناتج.

```java
options.setInputWorkingDirectory(new InputFileSystemDirectory("Your Input Directory"));
options.setOutputWorkingDirectory(new OutputFileSystemDirectory("Your Output Directory"));
```

*نصيحة:* استبدل `"Your Input Directory"` و `"Your Output Directory"` بالمسارات المطلقة أو النسبية الموجودة على جهازك.

## الخطوة 2: تكوين خيارات TeX

تتحكم هذه الخيارات في سلوك محرك LaTeX أثناء التحويل. اضبطها لتتناسب مع متطلبات مستندك.  
TeXOptions هي الفئة التي تحتفظ بهذه إعدادات التكوين.

```java
options.setInteraction(Interaction.NonstopMode);
options.setDateTime(new GregorianCalendar(2022, Calendar.DECEMBER, 18).getTime());
options.ignoreMissingPackages(true);
options.noLigatures(true);
options.repeat(true);
```

- **وضع التفاعل** – `NonstopMode` يستمر في المعالجة حتى إذا صادف أخطاء.  
- **DateTime** – يحدد التاريخ الذي يظهر في صفحة عنوان المستند.  
- **ignoreMissingPackages** – يمنع فشل العملية عندما لا يتم العثور على حزمة.  
- **noLigatures** – يعطل الروابط الطباعية إذا كنت تفضل الأحرف العادية.  
- **repeat** – يسمح للمحرك بإعادة التشغيل للمرجعيات المتقاطعة.

## الخطوة 3: تهيئة خيارات حفظ XPS

XpsSaveOptions يضبط معلمات إخراج XPS مثل الضغط وحجم الصفحة.

```java
options.setSaveOptions(new XpsSaveOptions());
```

## الخطوة 4: تخصيص خيارات حفظ XPS

ضبط دقيق لكيفية تعامل إخراج XPS مع الصيغ والرسومات والخطوط.  
`rasterizeFormulas` يحدد ما إذا كانت الصيغ تُعرض كصور.

```java
options.getSaveOptions().rasterizeFormulas(true);
options.getSaveOptions().rasterizeIncludedGraphics(true);
options.getSaveOptions().subsetFonts(true);
```

- **rasterizeFormulas** – يحول التعبيرات الرياضية إلى صور، مما يضمن عرضها بشكل صحيح على أي عارض XPS.  
- **rasterizeIncludedGraphics** – يجبر الرسومات المضمنة على التحويل إلى صور، مما قد يحسن التوافق.  
- **subsetFonts** – يضمّن فقط الحروف المستخدمة في المستند، مما يقلل حجم الملف بنسبة تصل إلى 40 % للفواتير النموذجية.

## الخطوة 5: تشغيل تحويل LaTeX إلى XPS

XpsDevice يكتب المحتوى المعروض إلى ملف XPS.  
`TeXJob` يربط ملف الإدخال، جهاز الإخراج (`XpsDevice`)، وجميع الخيارات التي قمت بتكوينها.

```java
new TeXJob("Your Input Directory" + "sample.ltx", new XpsDevice(), options).run();
```

بعد انتهاء استدعاء `run()`، ستجد `sample.xps` في مجلد الإخراج الذي حددته مسبقًا. يمكن إرسال ملف XPS هذا مباشرة إلى الطابعة أو تضمينه في حزمة فواتير PDF.

## أمثلة إضافية

### استخدام InputStream (تحويل سلسلة LaTeX مباشرةً)

إذا كنت تفضل إمداد مصدر LaTeX من الذاكرة بدلاً من ملف، غلف المصدر في `ByteArrayInputStream`.

```java
new TeXJob(new ByteArrayInputStream(
    "\\documentclass{article} \\begin{document} Hello, World! \\end{document}".getBytes("ASCII")),
    new XpsDevice(), options).run();
```

### استخدام الطرفية الرئيسية للإدخال (دع Aspose.TeX يحدد الملف تلقائيًا)

عندما لا تحتاج إلى تحديد مسار الملف صراحةً، يمكنك الاعتماد على الطرفية الافتراضية للإدخال.

```java
new TeXJob(new XpsDevice(), options).run();
```
