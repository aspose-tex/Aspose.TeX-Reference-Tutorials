يسمح لك استخدام تدفق خارجي بكتابة ناتج XPS مباشرةً إلى ذاكرة مؤقتة، نظام ملفات، أو مقبس شبكة دون إنشاء ملفات وسيطة. هذا النهج يحسن الأداء، يقلل من عبء I/O، ويسهل دمج التحويل في خدمات الويب أو خطوط الأنابيب السحابية.
