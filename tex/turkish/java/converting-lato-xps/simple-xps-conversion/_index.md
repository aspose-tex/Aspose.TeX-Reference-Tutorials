## Paketleri İçe Aktar

`import` bloğu gerekli sınıfları kapsam içine getirir. Bu tam listeyi korumak, kodun değişiklik yapmadan derlenmesini sağlar.

```java
package com.aspose.tex.LaTeXXpsConversionSimplest;

import java.io.ByteArrayInputStream;
import java.io.IOException;
import java.util.Calendar;
import java.util.GregorianCalendar;

import com.aspose.tex.InputFileSystemDirectory;
import com.aspose.tex.Interaction;
import com.aspose.tex.OutputFileSystemDirectory;
import com.aspose.tex.TeXConfig;
import com.aspose.tex.TeXJob;
import com.aspose.tex.TeXOptions;
import com.aspose.tex.rendering.XpsDevice;
import com.aspose.tex.rendering.XpsSaveOptions;

import util.Utils;
```

Şimdi dönüşüm adımlarını inceleyerek her kod parçasının amacını açıklayalım.

## Adım 1: Giriş ve Çıkış Dizinlerini Ayarla

Aspose.TeX'in kaynak `.ltx` dosyasını nerede bulacağını ve oluşturulan XPS dosyasını nereye yazacağını belirtmeniz gerekir.

```java
options.setInputWorkingDirectory(new InputFileSystemDirectory("Your Input Directory"));
options.setOutputWorkingDirectory(new OutputFileSystemDirectory("Your Output Directory"));
```

*İpucu:* `"Your Input Directory"` ve `"Your Output Directory"` ifadelerini, makinenizde mevcut olan mutlak ya da göreli yollarla değiştirin.

## Adım 2: TeX Seçeneklerini Yapılandır

Bu seçenekler, dönüşüm sırasında LaTeX motorunun nasıl davranacağını kontrol eder. Belgenizin gereksinimlerine göre ayarlayın.  
TeXOptions, bu yapılandırma ayarlarını tutan sınıftır.

```java
options.setInteraction(Interaction.NonstopMode);
options.setDateTime(new GregorianCalendar(2022, Calendar.DECEMBER, 18).getTime());
options.ignoreMissingPackages(true);
options.noLigatures(true);
options.repeat(true);
```

- **Interaction mode** – `NonstopMode` hatalarla karşılaşsa bile işlemeye devam eder.  
- **DateTime** – Belge başlık sayfasında görünen tarihi ayarlar.  
- **ignoreMissingPackages** – Bir paket bulunamadığında işin başarısız olmasını önler.  
- **noLigatures** – Düz karakterleri tercih ediyorsanız tipografik ligatürleri devre dışı bırakır.  
- **repeat** – Çapraz referanslar için motorun yeniden çalıştırılmasına izin verir.

## Adım 3: XPS Kaydetme Seçeneklerini Başlat

```java
options.setSaveOptions(new XpsSaveOptions());
```

## Adım 4: XPS Kaydetme Seçeneklerini Özelleştir

XPS çıktısının formüller, grafikler ve fontlarla nasıl başa çıkacağını ince ayar yapın.  
`rasterizeFormulas`, formüllerin görüntü olarak render edilip edilmediğini belirler.

```java
options.getSaveOptions().rasterizeFormulas(true);
options.getSaveOptions().rasterizeIncludedGraphics(true);
options.getSaveOptions().subsetFonts(true);
```

- **rasterizeFormulas** – Matematiksel ifadeleri görüntülere dönüştürür, böylece herhangi bir XPS görüntüleyicide doğru şekilde gösterilir.  
- **rasterizeIncludedGraphics** – Gömülü grafikleri rasterleştirir, bu da uyumluluğu artırabilir.  
- **subsetFonts** – Belgede kullanılan yalnızca karakterleri gömer, tipik faturalar için dosya boyutunu %40’a kadar azaltır.

## Adım 5: LaTeX'ten XPS Dönüşümünü Çalıştır

XpsDevice, render edilmiş içeriği bir XPS dosyasına yazar.  
`TeXJob`, giriş dosyasını, çıkış cihazını (`XpsDevice`) ve yapılandırdığınız tüm seçenekleri bir araya getirir.

```java
new TeXJob("Your Input Directory" + "sample.ltx", new XpsDevice(), options).run();
```

`run()` çağrısı tamamlandığında, daha önce belirttiğiniz çıkış dizininde `sample.xps` dosyasını bulacaksınız. Bu XPS dosyası doğrudan bir yazıcıya gönderilebilir veya PDF fatura paketine gömülebilir.

## Ek Örnekler

### InputStream Kullan (LaTeX dizesini doğrudan dönüştür)

Bellekten bir dosya yerine LaTeX kaynağını beslemeyi tercih ediyorsanız, kaynağı bir `ByteArrayInputStream` içinde sarın.

```java
new TeXJob(new ByteArrayInputStream(
    "\\documentclass{article} \\begin{document} Hello, World! \\end{document}".getBytes("ASCII")),
    new XpsDevice(), options).run();
```

### Ana Giriş Terminali Kullan (Aspose.TeX'in dosyayı otomatik bulmasına izin ver)

Dosya yolunu açıkça belirtmenize gerek kalmadığında, varsayılan giriş terminaline güvenebilirsiniz.

```java
new TeXJob(new XpsDevice(), options).run();
```
