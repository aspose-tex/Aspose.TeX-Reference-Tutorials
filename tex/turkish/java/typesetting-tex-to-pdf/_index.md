## Sık Sorulan Sorular

**S: Bu yaklaşımı sunucusuz bir platformda TeX'ten PDF üretmek için kullanabilir miyim?**  
C: Evet. Aspose.TeX yalnızca akışlarla çalıştığı için, disk yazımının sınırlı olduğu AWS Lambda, Azure Functions veya Google Cloud Run gibi ortamlara mükemmel uyum sağlar.

**S: Aspose.TeX arşivleme için PDF/A uyumluluğunu destekliyor mu?**  
C: Kesinlikle. Harici akışları kullanmaya devam ederken `PdfSaveOptions` sınıfı aracılığıyla PDF/A çıktısını etkinleştirebilirsiniz.

**S: Host makinede yüklü olmayan özel yazı tiplerini nasıl gömebilirim?**  
C: Yazı tipi dosyalarını uygulama kaynaklarınıza ekleyin ve `FontFactory.register()` ile yükledikten sonra `\setmainfont{MyFont}` ile referans verin.

**S: Büyük bir TeX belgesinin yalnızca bir bölümünü dönüştürmenin bir yolu var mı?**  
C: Kaynağı ayrı `InputStream` bölümlerine ayırabilir, her birini bağımsız olarak dönüştürüp, gerekirse elde edilen PDF'leri birleştirebilirsiniz.

**S: Hangi Java sürümleri destekleniyor?**  
C: Aspose.TeX for Java, Java 8'den Java 21'e kadar, tüm LTS sürümlerini destekler.
