## Häufig gestellte Fragen

**Q: Wie konvertiere ich LaTeX in PDF in einer .NET Core‑Anwendung?**  
A: Installieren Sie das Aspose.TeX‑NuGet‑Paket, laden Sie Ihre `.tex`‑Datei in ein `Document`‑Objekt und rufen Sie `Save("output.pdf")` auf. Die Bibliothek übernimmt die gesamte TeX‑Verarbeitung intern.

**Q: Kann ich die PDF‑Ausgabe (Schriftarten, Ränder usw.) anpassen?**  
A: Ja. Aspose.TeX stellt Rendering‑Optionen wie `DocumentOptions`, `FontSettings` und `PageSetup` bereit, mit denen Sie das endgültige PDF feinabstimmen können.

**Q: Ist es möglich, LaTeX direkt in SVG für die Web‑Nutzung zu konvertieren?**  
A: Absolut. Verwenden Sie die Methode `Save("output.svg")` oder die Klasse `SvgSaveOptions`, um skalierbare Vektorgrafiken aus Ihrem LaTeX‑Quellcode zu erzeugen.

**Q: Muss ich eine LaTeX‑Distribution auf dem Server installieren?**  
A: Nein. Aspose.TeX ist eine reine .NET‑Bibliothek und benötigt keine externen TeX‑Engines, was die Bereitstellung einfach und leichtgewichtig macht.

**Q: Was ist, wenn ich ein benutzerdefiniertes TeX‑Format oder -Paket verwenden muss?**  
A: Sie können benutzerdefinierte Formatdateien oder zusätzliche Pakete über die `TeXEngine`‑Konfiguration vor dem Rendering registrieren.
