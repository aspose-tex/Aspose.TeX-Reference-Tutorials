## Häufig gestellte Fragen

**Q: Kann ich mehrere Gleichungen in einem einzigen PNG rendern?**  
A: Ja. Verketteln Sie LaTeX‑Strings oder verwenden Sie die `Document`‑Klasse, um mehrere Formeln vor dem Rendern anzuordnen.

**Q: Unterstützt Aspose.TeX andere Bildformate?**  
A: Absolut. Neben PNG können Sie JPEG, BMP, GIF und TIFF ausgeben, indem Sie die `ImageFormat`‑Eigenschaft ändern.

**Q: Wie kann ich die Rendering‑Geschwindigkeit für die Stapelverarbeitung verbessern?**  
A: Verwenden Sie dieselbe `Renderer`‑Instanz erneut und laden Sie erforderliche Schriften vorab; das reduziert den Overhead bei jeder Konvertierung.

**Q: Ist es möglich, LaTeX zu einem transparenten PNG zu rendern?**  
A: Setzen Sie die Hintergrundfarbe auf `Color.Transparent` in den Rendering‑Optionen, bevor Sie `Save` aufrufen.

**Q: Welche .NET‑Version wird benötigt?**  
A: Aspose.TeX unterstützt .NET Framework 4.5+, .NET Core 3.1+, .NET 5 und .NET 6.
