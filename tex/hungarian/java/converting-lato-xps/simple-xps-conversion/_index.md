## Csomagok importálása

Az `import` blokk behozza a szükséges osztályokat a láthatóságba. Ennek a pontos listának a megtartása biztosítja, hogy a kód módosítás nélkül leforduljon.

```java
package com.aspose.tex.LaTeXXpsConversionSimplest;

import java.io.ByteArrayInputStream;
import java.io.IOException;
import java.util.Calendar;
import java.util.GregorianCalendar;

import com.aspose.tex.InputFileSystemDirectory;
import com.aspose.tex.Interaction;
import com.aspose.tex.OutputFileSystemDirectory;
import com.aspose.tex.TeXConfig;
import com.aspose.tex.TeXJob;
import com.aspose.tex.TeXOptions;
import com.aspose.tex.rendering.XpsDevice;
import com.aspose.tex.rendering.XpsSaveOptions;

import util.Utils;
```

Most lépésről‑lépésre végigvezetjük a konverziót, megmagyarázva minden kódrészlet célját.

## 1. lépés: Bemeneti és kimeneti könyvtárak beállítása

Meg kell adnia az Aspose.TeX‑nek, hol találja a `.ltx` forrásfájlt, és hová írja a létrejövő XPS fájlt.

```java
options.setInputWorkingDirectory(new InputFileSystemDirectory("Your Input Directory"));
options.setOutputWorkingDirectory(new OutputFileSystemDirectory("Your Output Directory"));
```

*Tip:* Cserélje le a `"Your Input Directory"` és `"Your Output Directory"` értékeket abszolút vagy relatív útvonalakra, amelyek léteznek a gépén.

## 2. lépés: TeX beállítások konfigurálása

Ezek a beállítások szabályozzák, hogyan viselkedik a LaTeX motor a konverzió során. Igazítsa őket a dokumentum igényeihez.  
A TeXOptions az a osztály, amely ezeket a konfigurációs beállításokat tartalmazza.

```java
options.setInteraction(Interaction.NonstopMode);
options.setDateTime(new GregorianCalendar(2022, Calendar.DECEMBER, 18).getTime());
options.ignoreMissingPackages(true);
options.noLigatures(true);
options.repeat(true);
```

- **Interaction mode** – `NonstopMode` folytatja a feldolgozást akkor is, ha hibákat talál.  
- **DateTime** – Beállítja a dokumentum címoldalán megjelenő dátumot.  
- **ignoreMissingPackages** – Megakadályozza, hogy a feladat hibával leálljon, ha egy csomag nem található.  
- **noLigatures** – Letiltja a tipográfiai ligatúrákat, ha egyszerű karaktereket szeretne.  
- **repeat** – Lehetővé teszi a motor számára, hogy újra fusson a kereszt‑hivatkozásokhoz.  

## 3. lépés: XPS mentési beállítások inicializálása

Az XpsSaveOptions konfigurálja az XPS kimenet paramétereit, például a tömörítést és az oldalméretet.

```java
options.setSaveOptions(new XpsSaveOptions());
```

## 4. lépés: XPS mentési beállítások testreszabása

Finomhangolja, hogyan kezeli az XPS kimenet a képleteket, grafikákat és betűkészleteket.  
A `rasterizeFormulas` határozza meg, hogy a képletek képként legyenek-e renderelve.

```java
options.getSaveOptions().rasterizeFormulas(true);
options.getSaveOptions().rasterizeIncludedGraphics(true);
options.getSaveOptions().subsetFonts(true);
```

- **rasterizeFormulas** – Matematikai kifejezéseket képekké konvertál, biztosítva, hogy bármely XPS megjelenítőn helyesen jelenjenek meg.  
- **rasterizeIncludedGraphics** – Kényszeríti a beágyazott grafikák rasterizálását, ami javíthatja a kompatibilitást.  
- **subsetFonts** – Csak a dokumentumban használt glifeket ágyazza be, így a fájlméret akár 40 %-kal is csökkenhet tipikus számlák esetén.  

## 5. lépés: LaTeX‑ról XPS‑re konverzió futtatása

Az XpsDevice a renderelt tartalmat XPS fájlba írja.  
A `TeXJob` összekapcsolja a bemeneti fájlt, a kimeneti eszközt (`XpsDevice`) és az összes beállítást, amelyet konfigurált.

```java
new TeXJob("Your Input Directory" + "sample.ltx", new XpsDevice(), options).run();
```

A `run()` hívás befejezése után megtalálja a `sample.xps` fájlt a korábban megadott kimeneti könyvtárban. Ez az XPS fájl közvetlenül elküldhető nyomtatóra, vagy beágyazható egy PDF számla csomagba.

## További példák

### InputStream használata (LaTeX karakterlánc közvetlen konvertálása)

Ha a LaTeX forrást memóriából szeretné betáplálni egy fájl helyett, csomagolja be egy `ByteArrayInputStream`‑be.

```java
new TeXJob(new ByteArrayInputStream(
    "\\documentclass{article} \\begin{document} Hello, World! \\end{document}".getBytes("ASCII")),
    new XpsDevice(), options).run();
```

### Fő bemeneti terminál használata (Az Aspose.TeX automatikusan megtalálja a fájlt)

Ha nem kell explicit módon megadnia a fájl útvonalát, támaszkodhat az alapértelmezett bemeneti terminálra.

```java
new TeXJob(new XpsDevice(), options).run();
```
