## Gyakran Ismételt Kérdések

**Q: Renderelhetek több egyenletet egyetlen PNG‑ben?**  
A: Igen. Fűzd össze a LaTeX karakterláncokat, vagy használd a `Document` osztályt több képlet elrendezéséhez a renderelés előtt.

**Q: Támogatja az Aspose.TeX más képformátumokat is?**  
A: Természetesen. A PNG mellett JPEG, BMP, GIF és TIFF formátumok is elérhetők az `ImageFormat` tulajdonság módosításával.

**Q: Hogyan javíthatom a renderelés sebességét kötegelt feldolgozás esetén?**  
A: Használd újra ugyanazt a `Renderer` példányt, és előtöltsd a szükséges betűtípusokat; ez csökkenti az egyes konverziók terhelését.

**Q: Lehetséges LaTeX‑et átlátszó PNG‑be renderelni?**  
A: Állítsd a háttérszínt `Color.Transparent`‑re a renderelési beállításokban, mielőtt meghívod a `Save`‑et.

**Q: Milyen .NET verzió szükséges?**  
A: Az Aspose.TeX támogatja a .NET Framework 4.5+, .NET Core 3.1+, .NET 5 és .NET 6 verziókat.
