## Vanliga frågor

**Q: Hur konverterar jag LaTeX till PDF i en .NET Core‑applikation?**  
A: Installera Aspose.TeX‑NuGet‑paketet, läs in din `.tex`‑fil i ett `Document`‑objekt och anropa `Save("output.pdf")`. Biblioteket hanterar all TeX‑bearbetning internt.

**Q: Kan jag anpassa PDF‑utdata (typsnitt, marginaler osv.)?**  
A: Ja. Aspose.TeX exponerar renderingsalternativ som `DocumentOptions`, `FontSettings` och `PageSetup` som låter dig finjustera den slutliga PDF‑filen.

**Q: Är det möjligt att konvertera LaTeX direkt till SVG för webbbruk?**  
A: Absolut. Använd metoden `Save("output.svg")` eller klassen `SvgSaveOptions` för att generera skalbara vektorgrafik från din LaTeX‑källa.

**Q: Behöver jag installera en LaTeX‑distribution på servern?**  
A: Nej. Aspose.TeX är ett rent .NET‑bibliotek och förlitar sig inte på externa TeX‑motorer, vilket gör distribution enkel och lättviktig.

**Q: Vad händer om jag behöver använda ett anpassat TeX‑format eller paket?**  
A: Du kan registrera egna formatfiler eller ytterligare paket via `TeXEngine`‑konfigurationen innan rendering.
