## Vanliga frågor

**Q: How do I **convert latex to pdf** using Aspose.TeX?**  
A: Instansiera ett `TeXDocument`, ladda din `.tex`-källa och anropa `Save("output.pdf")`. Samma API låter dig anropa `Save("output.png")` eller `Save("output.svg")` för andra format.

**Q: Can I **export latex as png** with custom resolution?**  
A: Ja. Använd `PngSaveOptions`-objektet för att ange DPI, bakgrundsfärg och bildkvalitet innan du sparar.

**Q: What is the best way to **generate pdf from latex** in a web service?**  
A: Distribuera konverteringskoden i ett stateless .NET Core API, cachea den resulterande PDF-filen när det är möjligt, och strömma filen tillbaka till klienten.

**Q: Is there a limit on the size of the LaTeX source I can convert?**  
A: Aspose.TeX hanterar stora dokument, men för extremt stora filer bör du överväga att öka minnesgränsen eller bearbeta dokumentet i sektioner.

**Q: Does Aspose.TeX support **convert latex to svg** for vector graphics?**  
A: Absolut. Använd `Save("output.svg")` eller `SvgSaveOptions`-klassen för att finjustera resultatet.
