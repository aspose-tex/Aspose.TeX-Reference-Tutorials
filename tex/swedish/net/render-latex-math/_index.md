## Vanliga frågor

**Q: Kan jag rendera flera ekvationer i en enda PNG?**  
A: Ja. Konkatenera LaTeX‑strängar eller använd `Document`‑klassen för att layouta flera formler innan rendering.

**Q: Stöder Aspose.TeX andra bildformat?**  
A: Absolut. Förutom PNG kan du exportera JPEG, BMP, GIF och TIFF genom att ändra `ImageFormat`‑egenskapen.

**Q: Hur förbättrar jag renderingshastigheten för batch‑bearbetning?**  
A: Återanvänd samma `Renderer`‑instans och för‑ladda nödvändiga typsnitt; detta minskar overhead för varje konvertering.

**Q: Är det möjligt att rendera LaTeX till en transparent PNG?**  
A: Sätt bakgrundsfärgen till `Color.Transparent` i renderingsalternativen innan du anropar `Save`.

**Q: Vilken version av .NET krävs?**  
A: Aspose.TeX stöder .NET Framework 4.5+, .NET Core 3.1+, .NET 5 och .NET 6.
