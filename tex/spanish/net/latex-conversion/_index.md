## Preguntas frecuentes

**Q: ¿Cómo **convertir latex a pdf** usando Aspose.TeX?**  
A: Instancie un `TeXDocument`, cargue su fuente `.tex` y llame a `Save("output.pdf")`. La misma API le permite llamar a `Save("output.png")` o `Save("output.svg")` para otros formatos.

**Q: ¿Puedo **exportar latex como png** con resolución personalizada?**  
A: Sí. Utilice el objeto `PngSaveOptions` para especificar DPI, color de fondo y calidad de imagen antes de guardar.

**Q: ¿Cuál es la mejor manera de **generar pdf from latex** en un servicio web?**  
A: Despliegue el código de conversión en una API sin estado de .NET Core, almacene en caché el PDF resultante cuando sea posible y transmita el archivo al cliente.

**Q: ¿Existe un límite en el tamaño del origen LaTeX que puedo convertir?**  
A: Aspose.TeX maneja documentos grandes, pero para archivos extremadamente extensos considere aumentar el límite de memoria o procesar el documento en secciones.

**Q: ¿Aspose.TeX admite **convertir latex a svg** para gráficos vectoriales?**  
A: Absolutamente. Use `Save("output.svg")` o la clase `SvgSaveOptions` para afinar la salida.
