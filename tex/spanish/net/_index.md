## Preguntas frecuentes

**Q: ¿Cómo convierto LaTeX a PDF en una aplicación .NET Core?**  
A: Instala el paquete NuGet Aspose.TeX, carga tu archivo `.tex` en un objeto `Document` y llama a `Save("output.pdf")`. La biblioteca maneja todo el procesamiento TeX internamente.

**Q: ¿Puedo personalizar la salida PDF (fuentes, márgenes, etc.)?**  
A: Sí. Aspose.TeX expone opciones de renderizado como `DocumentOptions`, `FontSettings` y `PageSetup` que te permiten afinar el PDF final.

**Q: ¿Es posible convertir LaTeX directamente a SVG para uso web?**  
A: Absolutamente. Usa el método `Save("output.svg")` o la clase `SvgSaveOptions` para generar gráficos vectoriales escalables desde tu código LaTeX.

**Q: ¿Necesito instalar una distribución LaTeX en el servidor?**  
A: No. Aspose.TeX es una biblioteca .NET pura y no depende de motores TeX externos, lo que simplifica la implementación.

**Q: ¿Qué pasa si necesito usar un formato o paquete TeX personalizado?**  
A: Puedes registrar archivos de formato personalizados o paquetes adicionales mediante la configuración `TeXEngine` antes de renderizar.
