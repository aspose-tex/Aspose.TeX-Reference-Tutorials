## ¿Qué es “generar SVG a partir de LaTeX”?
Generar SVG a partir de LaTeX significa convertir una expresión matemática formateada en LaTeX en un archivo Scalable Vector Graphics (SVG). SVG es independiente de la resolución, liviano y perfecto para renderizado web o de escritorio, lo que lo hace ideal para mostrar fórmulas complejas con claridad pixel‑perfecta.

## Preguntas Frecuentes

**P: ¿Puedo usar los archivos SVG generados en la web sin conversión adicional?**  
R: Sí—SVG es compatible de forma nativa con todos los navegadores modernos, por lo que puede incrustar la salida directamente en HTML o CSS.

**P: ¿Cómo cambio la fuente predeterminada para las matemáticas renderizadas?**  
R: Use la propiedad `FontFamily` de la configuración `TeXRenderer` para especificar cualquier fuente TrueType/OpenType instalada.

**P: ¿Es posible renderizar ecuaciones LaTeX que incluyan color o macros personalizadas?**  
R: Absolutamente. Aspose.TeX procesa paquetes de color estándar de LaTeX y le permite definir macros mediante el método `AddMacro`.

**P: ¿Qué tamaño tendrá el SVG generado?**  
R: Las dimensiones del SVG se calculan automáticamente en función del cuadro delimitador de la ecuación, pero puede sobrescribirlas usando los ajustes `Width` y `Height`.

**P: ¿La biblioteca admite el procesamiento por lotes de múltiples ecuaciones?**  
R: Sí—puede iterar a través de una colección de cadenas LaTeX y renderizar cada una a su propio archivo SVG con un sobrecosto mínimo.
