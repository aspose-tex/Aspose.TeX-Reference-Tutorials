## ¿Cómo generar PDF desde LaTeX en Java?
Cuando preguntas **cómo convertir LaTeX** a un PDF, la respuesta se reduce a tres pasos simples:

1. **Preparar la fuente LaTeX** – ya sea como una cadena en memoria o como un archivo `.tex` en disco.  
2. **Crear una instancia de `TeXDocument`** desde la API Aspose.TeX y cargar la fuente.  
3. **Llamar al método `save`** especificando `SaveFormat.PDF` para generar el archivo PDF final.

> **Consejo profesional:** Mantén tu fuente LaTeX codificada en UTF‑8 para evitar problemas de juego de caracteres, especialmente al manejar símbolos no ASCII.
