## よくある質問

**Q: Aspose.TeX を使用して **convert latex to pdf** はどうやって行いますか？**  
A: `TeXDocument` をインスタンス化し、`.tex` ソースをロードして `Save("output.pdf")` を呼び出します。同じ API で `Save("output.png")` や `Save("output.svg")` も呼び出すことができ、他のフォーマットにも対応します。

**Q: カスタム解像度で **export latex as png** は可能ですか？**  
A: はい。`PngSaveOptions` オブジェクトを使用して、保存前に DPI、背景色、画像品質を指定できます。

**Q: Web サービスで **generate pdf from latex** を行う最適な方法は何ですか？**  
A: ステートレスな .NET Core API に変換コードをデプロイし、可能な限り生成された PDF をキャッシュし、クライアントにストリーミングで返します。

**Q: 変換できる LaTeX ソースのサイズに制限はありますか？**  
A: Aspose.TeX は大きなドキュメントを処理できますが、極端に大きなファイルの場合はメモリ制限を増やすか、ドキュメントをセクションに分割して処理することを検討してください。

**Q: ベクターグラフィック用に **convert latex to svg** をサポートしていますか？**  
A: もちろんです。`Save("output.svg")` または `SvgSaveOptions` クラスを使用して出力を細かく調整できます。
