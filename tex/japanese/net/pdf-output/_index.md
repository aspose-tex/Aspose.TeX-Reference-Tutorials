## なぜ Aspose.TeX を PDF 生成に使用するのか？

- **High fidelity** – 出力 PDF は元の TeX ソースのすべての組版上のニュアンスを保持します。  
- **Cross‑platform** – Windows、Linux、macOS の .NET ランタイム上で一貫して動作します。  
- **No external dependencies** – TeX ディストリビューションやコマンドラインツールは不要です。  

## Aspose.TeX を使用して TeX から PDF を生成する方法

1. **Add the Aspose.TeX NuGet package** を .NET プロジェクトに追加します。  
2. **Create a `TeXDocument`** インスタンスを作成し、`.tex` ソースをロードします。  
3. **Call the `Save` method** で `PdfSaveOptions` を指定し、最終的な PDF ファイルを生成します。

## よくある質問

**Q: Aspose.TeX を商用アプリケーションで使用できますか？**  
A: はい、有効な Aspose ライセンスが必要です。評価用に無料トライアルが利用可能です。

**Q: Aspose.TeX はカスタム LaTeX パッケージをサポートしていますか？**  
A: ほとんどの標準パッケージはデフォルトでサポートされています。珍しいパッケージについては、パーサーを拡張することで対応可能です。

**Q: 大規模な TeX 文書を扱う最適な方法は何ですか？**  
A: 文書をセクションごとに処理し、PDF 出力をストリーミングすることでメモリ使用量を抑えます。

**Q: レンダリングの問題をデバッグするには？**  
A: ライブラリのロギング機能を有効にして、詳細なパース情報を取得します。

**Q: 生成された PDF にフォントを埋め込むことは可能ですか？**  
A: はい、`PdfSaveOptions` の `EmbedFonts` オプションを設定して、使用したすべてのフォントを埋め込みます。
