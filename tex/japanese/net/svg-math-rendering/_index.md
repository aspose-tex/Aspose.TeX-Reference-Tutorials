## 「LaTeX から SVG を生成」とは何ですか？

LaTeX から SVG を生成するとは、LaTeX 形式の数式表現を Scalable Vector Graphics (SVG) ファイルに変換することを意味します。SVG は解像度に依存せず、軽量で、Web やデスクトップでのレンダリングに最適であり、ピクセル単位で完璧な明瞭さで複雑な数式を表示するのに理想的です。

## よくある質問

**Q: 生成した SVG ファイルを追加変換なしでウェブで使用できますか？**  
A: はい — SVG はすべての最新ブラウザでネイティブにサポートされているため、出力を直接 HTML や CSS に埋め込むことができます。

**Q: レンダリングされた数式のデフォルトフォントを変更するにはどうすればよいですか？**  
A: `TeXRenderer` の設定の `FontFamily` プロパティを使用して、インストールされている任意の TrueType/OpenType フォントを指定します。

**Q: 色やカスタムマクロを含む LaTeX 方程式をレンダリングすることは可能ですか？**  
A: もちろんです。Aspose.TeX は標準の LaTeX カラーパッケージを処理し、`AddMacro` メソッドを使用してマクロを定義することができます。

**Q: 生成される SVG のサイズはどれくらいですか？**  
A: SVG の寸法は数式のバウンディングボックスに基づいて自動的に計算されますが、`Width` と `Height` 設定を使用して上書きすることも可能です。

**Q: ライブラリは複数の方程式のバッチ処理をサポートしていますか？**  
A: はい — LaTeX 文字列のコレクションをループし、各々を最小のオーバーヘッドで個別の SVG ファイルにレンダリングできます。
