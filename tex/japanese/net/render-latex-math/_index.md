## Frequently Asked Questions

**Q: Can I render multiple equations in a single PNG?**  
A: Yes. Concatenate LaTeX strings or use the `Document` class to layout several formulas before rendering.

**Q: Does Aspose.TeX support other image formats?**  
A: Absolutely. Besides PNG, you can output JPEG, BMP, GIF, and TIFF by changing the `ImageFormat` property.

**Q: How do I improve rendering speed for batch processing?**  
A: Reuse the same `Renderer` instance and pre‑load required fonts; this reduces overhead for each conversion.

**Q: Is it possible to render LaTeX to a transparent PNG?**  
A: Set the background color to `Color.Transparent` in the rendering options before calling `Save`.

**Q: What version of .NET is required?**  
A: Aspose.TeX supports .NET Framework 4.5+, .NET Core 3.1+, .NET 5, and .NET 6.
