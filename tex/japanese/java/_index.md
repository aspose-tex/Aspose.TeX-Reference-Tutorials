### Aspose.TeX とは？

Aspose.TeX は、TeX/LaTeX ソースファイルを解析し、コードから直接 PDF、XPS、PNG、SVG などの形式を生成する Java ライブラリです。TeX エンジンの複雑さを抽象化し、開発者がエンタープライズアプリケーションに科学出版ワークフローを組み込むためのシンプルな API を提供します。

## JavaでLaTeXをPDFに変換する方法は？

`TeXDocument` クラスは LaTeX ソースファイルを表し、ロード、処理、レンダリングのメソッドを提供します。  
`save` は、処理されたドキュメントを PDF などの選択した出力形式に書き込みます。

`TeXDocument` クラスで `.tex` ソースをロードし、`save("output.pdf", SaveFormat.Pdf)` を呼び出すだけで、2 行のシーケンスで完全な **how to convert latex** 操作が実行されます。ライブラリはパッケージ、フォント、画像を自動的に解決し、元の LaTeX レイアウトと一致する PDF を生成します。大規模なドキュメントの場合、ストリーミングモードを有効にすると、500 MB 超のファイルでもメモリ使用量を 100 MB 未満に抑えることができます。

### なぜ Aspose.TeX for Java を使用するのか？

- **広範なフォーマットサポート** – PDF、XPS、PNG、SVG など、30 以上の入力および出力フォーマットに対応。  
- **高性能処理** – 標準サーバー上で 200 ページの LaTeX ドキュメントを 2 秒未満でレンダリング可能。  
- **外部 TeX のインストール不要** – 完全にオフラインで動作し、クラウドやコンテナ環境に最適です。
