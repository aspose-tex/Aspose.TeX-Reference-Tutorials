## Cara Menghasilkan PDF dari LaTeX di Java?
Saat Anda menanyakan **how to convert LaTeX** menjadi PDF, jawabannya dapat diringkas dalam tiga langkah sederhana:

1. **Siapkan sumber LaTeX** – baik sebagai string di memori atau sebagai file `.tex` di disk.  
2. **Buat instance `TeXDocument`** dari API Aspose.TeX dan muat sumbernya.  
3. **Panggil metode `save`** dengan menentukan `SaveFormat.PDF` untuk menghasilkan file PDF akhir.

> **Pro tip:** Pastikan sumber LaTeX Anda dienkode UTF‑8 untuk menghindari masalah set karakter, terutama saat menangani simbol non‑ASCII.
