## Domande Frequenti

**Q: Posso usare i file SVG generati sul web senza conversioni aggiuntive?**  
A: Sì—l'SVG è supportato nativamente da tutti i browser moderni, quindi potete incorporare l'output direttamente in HTML o CSS.

**Q: Come cambio il carattere predefinito per la matematica renderizzata?**  
A: Utilizzate la proprietà `FontFamily` della configurazione `TeXRenderer` per specificare qualsiasi font TrueType/OpenType installato.

**Q: È possibile renderizzare equazioni LaTeX che includono colore o macro personalizzate?**  
A: Assolutamente. Aspose.TeX elabora i pacchetti colore standard di LaTeX e consente di definire macro tramite il metodo `AddMacro`.

**Q: Quale dimensione avrà l'SVG generato?**  
A: Le dimensioni dell'SVG sono calcolate automaticamente in base al bounding box dell'equazione, ma potete sovrascriverle usando le impostazioni `Width` e `Height`.

**Q: La libreria supporta l'elaborazione batch di più equazioni?**  
A: Sì—potete iterare su una collezione di stringhe LaTeX e renderizzare ciascuna in un proprio file SVG con un overhead minimo.
