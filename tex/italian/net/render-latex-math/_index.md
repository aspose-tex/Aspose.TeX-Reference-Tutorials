## Domande frequenti

**Q: Posso renderizzare più equazioni in un unico PNG?**  
A: Sì. Concatenare le stringhe LaTeX o usare la classe `Document` per disporre più formule prima del rendering.

**Q: Aspose.TeX supporta altri formati immagine?**  
A: Assolutamente. Oltre al PNG, puoi esportare JPEG, BMP, GIF e TIFF modificando la proprietà `ImageFormat`.

**Q: Come miglioro la velocità di rendering per l'elaborazione batch?**  
A: Riutilizza la stessa istanza `Renderer` e pre‑carica i font necessari; questo riduce l'overhead per ogni conversione.

**Q: È possibile renderizzare LaTeX in un PNG trasparente?**  
A: Imposta il colore di sfondo su `Color.Transparent` nelle opzioni di rendering prima di chiamare `Save`.

**Q: Quale versione di .NET è richiesta?**  
A: Aspose.TeX supporta .NET Framework 4.5+, .NET Core 3.1+, .NET 5 e .NET 6.
