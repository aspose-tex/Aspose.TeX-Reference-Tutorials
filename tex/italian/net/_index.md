## Domande Frequenti

**Q: Come converto LaTeX in PDF in un'applicazione .NET Core?**  
A: Installa il pacchetto NuGet Aspose.TeX, carica il tuo file `.tex` in un oggetto `Document` e chiama `Save("output.pdf")`. La libreria gestisce internamente tutta l'elaborazione TeX.

**Q: Posso personalizzare l'output PDF (font, margini, ecc.)?**  
A: Sì. Aspose.TeX espone opzioni di rendering come `DocumentOptions`, `FontSettings` e `PageSetup` che ti permettono di regolare finemente il PDF finale.

**Q: È possibile convertire LaTeX direttamente in SVG per uso web?**  
A: Assolutamente. Usa il metodo `Save("output.svg")` o la classe `SvgSaveOptions` per generare grafica vettoriale scalabile dal tuo sorgente LaTeX.

**Q: Devo installare una distribuzione LaTeX sul server?**  
A: No. Aspose.TeX è una libreria .NET pura e non dipende da motori TeX esterni, rendendo la distribuzione semplice e leggera.

**Q: Cosa succede se devo usare un formato TeX o un pacchetto personalizzato?**  
A: Puoi registrare file di formato personalizzati o pacchetti aggiuntivi tramite la configurazione `TeXEngine` prima del rendering.
