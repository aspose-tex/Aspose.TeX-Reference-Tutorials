## Come generare PDF da LaTeX in Java?

Quando ti chiedi **come convertire LaTeX** in un PDF, la risposta si riduce a tre semplici passaggi:

1. **Prepara il sorgente LaTeX** – sia come stringa in memoria sia come file `.tex` su disco.  
2. **Crea un'istanza di `TeXDocument`** dall'API Aspose.TeX e carica il sorgente.  
3. **Chiama il metodo `save`** specificando `SaveFormat.PDF` per produrre il file PDF finale.  

> **Consiglio professionale:** Mantieni il tuo sorgente LaTeX codificato in UTF‑8 per evitare problemi di set di caratteri, soprattutto quando lavori con simboli non‑ASCII.
