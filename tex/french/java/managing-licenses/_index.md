## Questions fréquentes

**Q: Puis‑je passer d'une licence basée sur un fichier à une licence à consommation mesurée sans redéployer l'application ?**  
A: Oui. Remplacez simplement le code d'initialisation de la licence par l'appel à la licence à consommation mesurée et redémarrez l'application.

**Q: Que se passe‑t‑il si le fichier de licence est manquant ou corrompu ?**  
A: Aspose.TeX lèvera une `LicenseException`. Capturez l'exception pour fournir un message d'erreur convivial ou une logique de secours.

**Q: Dois‑je définir la licence pour chaque thread dans un environnement multi‑threadé ?**  
A: Non. La licence est appliquée globalement une fois chargée ; tous les threads suivants en héritent.

**Q: Existe‑t‑il un moyen de vérifier que la licence a été chargée avec succès ?**  
A: Après avoir appelé `License.setLicense(...)`, vous pouvez appeler `License.isLicenseSet()` (ou vérifier les exceptions) pour confirmer une activation réussie.

**Q: Puis‑je utiliser le même fichier de licence à la fois sur des serveurs Windows et Linux ?**  
A: Absolument. Le fichier de licence est indépendant de la plateforme tant que le chemin d'accès est correct et accessible.
