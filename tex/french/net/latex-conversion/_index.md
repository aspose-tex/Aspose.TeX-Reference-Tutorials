## Frequently Asked Questions

**Q: Comment puis‑je **convert latex to pdf** avec Aspose.TeX?**  
R: Instanciez un `TeXDocument`, chargez votre source `.tex`, puis appelez `Save("output.pdf")`. La même API vous permet d’appeler `Save("output.png")` ou `Save("output.svg")` pour d’autres formats.

**Q: Puis‑je **export latex as png** avec une résolution personnalisée?**  
R: Oui. Utilisez l’objet `PngSaveOptions` pour spécifier le DPI, la couleur d’arrière‑plan et la qualité de l’image avant l’enregistrement.

**Q: Quelle est la meilleure façon de **generate pdf from latex** dans un service web?**  
R: Déployez le code de conversion dans une API .NET Core sans état, mettez en cache le PDF résultant lorsque c’est possible, et diffusez le fichier au client.

**Q: Existe‑t‑il une limite de taille pour le source LaTeX que je peux convertir?**  
R: Aspose.TeX gère les documents volumineux, mais pour des fichiers extrêmement grands, envisagez d’augmenter la limite de mémoire ou de traiter le document par sections.

**Q: Aspose.TeX prend‑il en charge **convert latex to svg** pour les graphiques vectoriels?**  
R: Absolument. Utilisez `Save("output.svg")` ou la classe `SvgSaveOptions` pour affiner la sortie.
