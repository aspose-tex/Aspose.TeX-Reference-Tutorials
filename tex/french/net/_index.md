## Questions fréquentes

**Q : Comment convertir LaTeX en PDF dans une application .NET Core ?**  
R : Installez le package NuGet Aspose.TeX, chargez votre fichier `.tex` dans un objet `Document`, puis appelez `Save("output.pdf")`. La bibliothèque gère tout le traitement TeX en interne.

**Q : Puis‑je personnaliser la sortie PDF (polices, marges, etc.) ?**  
R : Oui. Aspose.TeX expose des options de rendu telles que `DocumentOptions`, `FontSettings` et `PageSetup` qui vous permettent d’ajuster finement le PDF final.

**Q : Est‑il possible de convertir LaTeX directement en SVG pour le web ?**  
R : Absolument. Utilisez la méthode `Save("output.svg")` ou la classe `SvgSaveOptions` pour générer des graphiques vectoriels évolutifs à partir de votre source LaTeX.

**Q : Dois‑je installer une distribution LaTeX sur le serveur ?**  
R : Non. Aspose.TeX est une bibliothèque pure .NET et ne dépend d’aucun moteur TeX externe, ce qui rend le déploiement simple et léger.

**Q : Que faire si je dois utiliser un format ou un package TeX personnalisé ?**  
R : Vous pouvez enregistrer des fichiers de format personnalisés ou des packages supplémentaires via la configuration `TeXEngine` avant le rendu.
