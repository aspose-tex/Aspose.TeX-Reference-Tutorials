## FAQ

**Q : Puis‑je rendre plusieurs équations dans un même PNG ?**  
R : Oui. Concaténez les chaînes LaTeX ou utilisez la classe `Document` pour disposer plusieurs formules avant le rendu.

**Q : Aspose.TeX prend‑il en charge d’autres formats d’image ?**  
R : Absolument. En plus du PNG, vous pouvez obtenir JPEG, BMP, GIF et TIFF en modifiant la propriété `ImageFormat`.

**Q : Comment améliorer la vitesse de rendu pour le traitement par lots ?**  
R : Réutilisez la même instance de `Renderer` et pré‑chargez les polices nécessaires ; cela réduit la surcharge pour chaque conversion.

**Q : Est‑il possible de rendre LaTeX en PNG transparent ?**  
R : Définissez la couleur d’arrière‑plan sur `Color.Transparent` dans les options de rendu avant d’appeler `Save`.

**Q : Quelle version de .NET est requise ?**  
R : Aspose.TeX prend en charge .NET Framework 4.5+, .NET Core 3.1+, .NET 5 et .NET 6.
