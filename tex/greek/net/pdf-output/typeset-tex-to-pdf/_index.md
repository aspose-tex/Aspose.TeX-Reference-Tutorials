Η διαδικασία μετατροπής αναλύει το markup TeX, επιλύει τα macros, διατάσσει το έγγραφο και τελικά ρέει τις αποδομένες σελίδες σε ένα αρχείο PDF που μπορεί να ανοιχθεί σε οποιαδήποτε πλατφόρμα.

## Βήμα 1: Ρύθμιση Καταλόγων Εισόδου και Εξόδου

**Direct answer:** Δημιουργήστε δύο προσωρινά αρχεία ZIP – ένα για την πηγή TeX και τους πόρους (εισόδου) και ένα για το παραγόμενο PDF (εξόδου). Αυτή η προσέγγιση απομονώνει την εργασία, διευκολύνει τη ροή δεδομένων και λειτουργεί σταθερά σε όλες τις πλατφόρμες.

Χρησιμοποιούμε το `System.IO.Compression.ZipArchive` για να δημιουργήσουμε τα αρχεία ZIP στη μνήμη, αποφεύγοντας τυχόν παρενέργειες του συστήματος αρχείων.

```csharp
using Aspose.TeX.IO;
using Aspose.TeX.Presentation.Pdf;
using System.IO;
```

## Βήμα 2: Ορισμός Επιλογών Μετατροπής

**Direct answer:** Δημιουργήστε ένα αντικείμενο `TeXConversionOptions`, ορίστε το όνομα εργασίας ώστε να ταιριάζει με το αρχείο `.tex` σας (χωρίς επέκταση) και δείξτε του τα αρχεία ZIP εισόδου και εξόδου. Αυτό το αντικείμενο λέει στο Aspose.TeX από πού να διαβάσει και πού να γράψει το παραγόμενο PDF.

Το TeXConversionOptions περιλαμβάνει τη διαμόρφωση της εργασίας, συμπεριλαμβανομένων των αρχείων ZIP εισόδου και εξόδου και του ονόματος του αρχείου TeX που θα επεξεργαστεί.

`PdfDevice` είναι ο προορισμός απόδοσης που γράφει τα bytes PDF στη ροή εξόδου.

## Βήμα 3: Ορισμός Επιλογών Αποθήκευσης (save pdf with options)

**Direct answer:** Δημιουργήστε μια παρουσία `PdfSaveOptions` για να ελέγξετε τη συμπίεση, την ενσωμάτωση γραμματοσειρών και τα μεταδεδομένα πριν γραφτεί το PDF. Μπορείτε επίσης να ορίσετε το μέγεθος σελίδας, τα περιθώρια και την κρυπτογράφηση εδώ.

Το PdfSaveOptions ελέγχει πώς γράφεται το παραγόμενο PDF, όπως το επίπεδο συμπίεσης, η ενσωμάτωση γραμματοσειρών και τα μεταδεδομένα.

```csharp
// Define TeX conversion options
TeXOptions options = TeXOptions.ConsoleAppOptions(TeXConfig.ObjectTeX());
options.JobName = "typeset-pdf-to-external-stream";
options.InputWorkingDirectory = new InputZipDirectory(inZipStream, "in");
options.OutputWorkingDirectory = new OutputZipDirectory(outZipStream);
options.TerminalOut = new OutputFileTerminal(options.OutputWorkingDirectory);
```

## Βήμα 4: Τυπογραφία TeX σε PDF

**Direct answer:** Ανοίξτε μια ροή εγγραφής (π.χ., `FileStream` ή `MemoryStream`) για το PDF εξόδου, στη συνέχεια καλέστε `TeXProcessor.Typeset(options, device)`. Ο επεξεργαστής διαβάζει την πηγή TeX, τη μεταγλωττίζει και ρέει το τελικό PDF στη δοθείσα συσκευή.

Το TeXProcessor είναι η κύρια μηχανή που διαβάζει την πηγή TeX, εκτελεί τη μεταγλώττιση και παράγει την τελική έξοδο χρησιμοποιώντας τη συγκεκριμένη συσκευή.

Αυτό το βήμα εκτελεί την πραγματική λειτουργία **convert tex to pdf** και παράγει ένα έτοιμο προς χρήση αρχείο PDF.

## Βήμα 5: Ολοκλήρωση Εξόδου

**Direct answer:** Κλείστε το αρχείο ZIP εξόδου για να ολοκληρώσετε το πακέτο, στη συνέχεια εξάγετε το αρχείο PDF από το ZIP ή ρέστε το απευθείας στον πελάτη. Το κλείσιμο του ZIP εξασφαλίζει ότι όλες οι καταχωρήσεις έχουν εκσυγχρονιστεί και η δομή ZIP είναι έγκυρη.
