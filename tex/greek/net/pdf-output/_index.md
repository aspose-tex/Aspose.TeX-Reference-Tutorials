## Πώς να Δημιουργήσετε PDF από TeX Χρησιμοποιώντας το Aspose.TeX

1. **Προσθέστε το πακέτο NuGet Aspose.TeX** στο .NET project σας.  
2. **Δημιουργήστε ένα αντικείμενο `TeXDocument`** και φορτώστε την πηγή `.tex` σας.  
3. **Καλέστε τη μέθοδο `Save`** καθορίζοντας `PdfSaveOptions` για να δημιουργήσετε το τελικό αρχείο PDF.  

*(Το πραγματικό απόσπασμα κώδικα είναι διαθέσιμο στον συνδεδεμένο λεπτομερή οδηγό.)*
