# Προηγμένη Είσοδος και Έξοδος Aspose.TeX

## Εισαγωγή

Το Aspose.TeX for .NET είναι ένας αλλαγή‑παιχνιδιού στην απρόσκοπτη ενσωμάτωση TeX, παρέχοντας στους προγραμματιστές μια ισχυρή βιβλιοθήκη για τη βελτίωση της επεξεργασίας εγγράφων. Σε αυτό το άρθρο, εμβαθύνουμε σε προχωρημένα tutorials που εστιάζουν στον καθορισμό των καταλόγων εισόδου και στην κυριαρχία των streams, των εικόνων και της εισόδου τερματικού σε C#. **Αν ψάχνετε για το πώς να ορίσετε την είσοδο** για τα έργα σας σε TeX, βρίσκεστε στο σωστό μέρος.

## Γρήγορες Απαντήσεις
- **Τι σημαίνει το “how to set input”;**  
  Σημαίνει τη ρύθμιση της βιβλιοθήκης ώστε να εντοπίζει σωστά τα αρχεία πηγής TeX, τις εικόνες και τα δεδομένα stream.
- **Ποια κλάση API διαχειρίζεται τους καταλόγους εισόδου;**  
  Το `TeXInputOptions` σας επιτρέπει να ορίσετε το βασικό φάκελο και πρόσθετες διαδρομές αναζήτησης.
- **Μπορώ να προσθέσω εικόνες απευθείας από ένα stream;**  
  Ναι, χρησιμοποιώντας τη μέθοδο `AddImage` στις επιλογές εισόδου (δείτε “how to add images” παρακάτω).
- **Υποστηρίζεται η είσοδος τερματικού;**  
  Απόλυτα – μπορείτε να τροφοδοτήσετε κώδικα LaTeX μέσω ενός `MemoryStream` ή της τυπικής εισόδου.
- **Χρειάζομαι άδεια για χρήση σε παραγωγή;**  
  Απαιτείται έγκυρη άδεια Aspose.TeX για μη‑αξιολογικές εγκαταστάσεις.

## Πώς να Ορίσετε την Είσοδο στο Aspose.TeX για .NET
Η ρύθμιση του περιβάλλοντος εισόδου είναι η βάση κάθε ροής εργασίας Aspose.TeX. Παρακάτω θα βρείτε τα τρία πιο κοινά σενάρια:

### Πώς να Προσθέσετε Εικόνες με το Aspose.TeX
Οι εικόνες συχνά αναφέρονται σε αρχεία TeX χρησιμοποιώντας σχετικές διαδρομές. Με τη διαμόρφωση των επιλογών εισόδου μπορείτε να κατευθύνετε τη μηχανή σε έναν φάκελο που περιέχει όλα τα απαιτούμενα γραφικά, ή μπορείτε να παρέχετε ένα stream εικόνας απευθείας. Αυτό εξαλείφει την ανάγκη αντιγραφής αρχείων γύρω από το έργο σας.

### Πώς να Επεξεργαστείτε Streams στο Aspose.TeX
Όταν εργάζεστε με δυναμικά παραγόμενο περιεχόμενο LaTeX (π.χ., δημιουργώντας μια αναφορά εν κινήσει), θα θέλετε να τροφοδοτήσετε την πηγή ως stream αντί για φυσικό αρχείο. Το Aspose.TeX δέχεται οποιοδήποτε αντικείμενο `Stream`, επιτρέποντάς σας να ενσωματώσετε υπηρεσίες web, βάσεις δεδομένων ή γεννήτριες εν ενθύμηση.

### Πώς να Ορίσετε τον Κατάλογο Εισόδου
1. **Δημιουργήστε μια παρουσία του `TeXInputOptions`.**  
   Αυτό το αντικείμενο περιέχει όλες τις ρυθμίσεις σχετικές με διαδρομές.  
2. **Καθορίστε τον βασικό κατάλογο** όπου βρίσκεται το κύριο αρχείο `.tex`.  
3. **Προσθέστε πρόσθετες διαδρομές αναζήτησης** για υποφακέλους που περιέχουν εικόνες ή βοηθητικά αρχεία.  
4. **Περάστε τις διαμορφωμένες επιλογές** στο `TeXProcessor` πριν από τη δημιουργία.

Αυτά τα βήματα διασφαλίζουν ότι η βιβλιοθήκη μπορεί να εντοπίσει κάθε πόρο χωρίς χειροκίνητη αντιγραφή αρχείων, καθιστώντας τη διαδικασία κατασκευής πιο καθαρή και πιο συντηρήσιμη.

## Συχνές Ερωτήσεις

**Q: Μπορώ να αλλάξω τον κατάλογο εισόδου κατά την εκτέλεση;**  
A: Ναι, μπορείτε να δημιουργήσετε ένα νέο αντικείμενο `TeXInputOptions` και να το περάσετε στον επεξεργαστή όποτε χρειαστεί να επαναρυθμίσετε τη διαδρομή.

**Q: Πώς προσθέτω εικόνες που αποθηκεύονται σε βάση δεδομένων;**  
A: Ανακτήστε την εικόνα ως `byte[]`, τυλίξτε την σε `MemoryStream` και χρησιμοποιήστε τη μέθοδο `AddImage` στις επιλογές εισόδου (δείτε “how to add images”).

**Q: Είναι δυνατόν να επεξεργαστώ κώδικα LaTeX που λαμβάνεται από web API χωρίς αποθήκευση αρχείου;**  
A: Απόλυτα. Τροφοδοτήστε το ακατέργαστο string LaTeX σε `MemoryStream` και ορίστε το ως πηγή stream για τον επεξεργαστή (δείτε “how to process streams”).

**Q: Πρέπει να καλέσω κάποια μεθόδους καθαρισμού μετά την επεξεργασία;**  
A: Αποδεσμεύστε τυχόν streams που δημιουργήσατε και, αν επεξεργάζεστε μεγάλα έγγραφα, σκεφτείτε να καλέσετε `TeXProcessor.Cleanup()` για να ελευθερώσετε πόρους.

**Q: Πού μπορώ να βρω πιο προχωρημένα παραδείγματα;**  
A: Οι δύο παραπάνω σύνδεσμοι tutorial περιέχουν πλήρη δείγματα κώδικα που δείχνουν κάθε σενάριο με λεπτομέρεια.
