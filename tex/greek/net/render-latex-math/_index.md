## Συχνές Ερωτήσεις

**Q: Μπορώ να αποδώσω πολλαπλές εξισώσεις σε ένα μόνο PNG;**  
A: Ναι. Συγκεντρώστε συμβολοσειρές LaTeX ή χρησιμοποιήστε την κλάση `Document` για να τοποθετήσετε αρκετούς τύπους πριν την απόδοση.

**Q: Υποστηρίζει το Aspose.TeX άλλες μορφές εικόνας;**  
A: Απόλυτα. Εκτός από PNG, μπορείτε να εξάγετε JPEG, BMP, GIF και TIFF αλλάζοντας την ιδιότητα `ImageFormat`.

**Q: Πώς βελτιώνω την ταχύτητα απόδοσης για επεξεργασία σε παρτίδες;**  
A: Επαναχρησιμοποιήστε το ίδιο αντικείμενο `Renderer` και προφορτώστε τις απαιτούμενες γραμματοσειρές· αυτό μειώνει το φορτίο για κάθε μετατροπή.

**Q: Είναι δυνατόν να αποδώσω LaTeX σε διαφανές PNG;**  
A: Ορίστε το χρώμα φόντου σε `Color.Transparent` στις επιλογές απόδοσης πριν καλέσετε τη μέθοδο `Save`.

**Q: Ποια έκδοση του .NET απαιτείται;**  
A: Το Aspose.TeX υποστηρίζει .NET Framework 4.5+, .NET Core 3.1+, .NET 5 και .NET 6.
