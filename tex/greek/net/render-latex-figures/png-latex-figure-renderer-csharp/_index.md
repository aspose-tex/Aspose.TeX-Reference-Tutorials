## Οδηγός βήμα‑βήμα για την απόδοση LaTeX σε PNG

### Βήμα 1: Ρύθμιση επιλογών απόδοσης (FigureRendererOptions)

Δημιουργήστε ένα αντικείμενο `FigureRendererOptions` και ρυθμίστε την ανάλυση, το προ-εγγραφο, τον παράγοντα κλιμάκωσης, το χρώμα φόντου και τις επιλογές καταγραφής. Η ρύθμιση της **latex png resolution** εδώ επηρεάζει άμεσα την ευκρίνεια της τελικής εικόνας.

```csharp
FigureRendererOptions options = new PngFigureRendererOptions() { Resolution = 150 };
options.Preamble = "\\usepackage{pict2e}";
options.Scale = 3000;
options.BackgroundColor = System.Drawing.Color.White;
options.LogStream = new System.IO.MemoryStream();
options.ShowTerminal = true;
```

### Βήμα 2: Ορισμός ροής εξόδου και διαστάσεων

Προετοιμάστε μια ροή εξόδου όπου θα αποθηκευτεί το PNG και μια δομή `SizeF` για να λάβετε τις διαστάσεις της αποδοθείσας εικόνας. Εδώ είναι που **δημιουργείτε PNG από LaTeX** στο δίσκο.

System.Drawing.SizeF size = new System.Drawing.SizeF();
using (System.IO.Stream stream = System.IO.File.Open(

### Βήμα 3: Εκτέλεση της διαδικασίας απόδοσης

Περάστε το LaTeX source, τη ροή εξόδου, τις επιλογές που διαμορφώσατε και τη μεταβλητή μεγέθους στον renderer. Ο renderer μετατρέπει το περιβάλλον picture του LaTeX σε μια **υψηλής ποιότητας LaTeX PNG**.

```csharp
new PngFigureRenderer().Render(@"\setlength{\unitlength}{0.8cm}
\begin{picture}(6,5)
    % LaTeX figure code goes here
\end{picture}", stream, options, out size);
```

Μετά την απόδοση, εκτυπώστε τυχόν μηνύματα σφάλματος και το τελικό μέγεθος εικόνας στην κονσόλα. Αυτό σας βοηθά να επαληθεύσετε ότι η λειτουργία **render latex to png** ολοκληρώθηκε επιτυχώς.

## Υψηλής Ποιότητας LaTeX PNG: Ρύθμιση ανάλυσης και κλίμακας

Αν χρειάζεστε πιο οξεία εικόνα για εκτύπωση, αυξήστε το `Resolution` (π.χ., 300 dpi) ή τον παράγοντα `Scale`. Λάβετε υπόψη ότι μεγαλύτερες τιμές αυξάνουν τη χρήση μνήμης, οπότε δοκιμάστε τις ρυθμίσεις που ταιριάζουν καλύτερα στο περιβάλλον σας.

## Κοινά Προβλήματα και Λύσεις

| Πρόβλημα | Αιτία | Διόρθωση |
|----------|-------|----------|
| **Κενό PNG** | Η ροή εξόδου δεν έχει αδειάσει ή κλείσει | Βεβαιωθείτε ότι το `using` block ολοκληρώνεται πριν διαβάσετε το αρχείο. |
| **Λείπουν πακέτα** | Ο κώδικας LaTeX χρησιμοποιεί πακέτο που δεν υπάρχει στο προ-εγγραφο | Προσθέστε το απαιτούμενο `\usepackage{...}` στο `options.Preamble`. |
| **Χαμηλή ανάλυση** | Η προεπιλεγμένη DPI είναι πολύ χαμηλή για εκτύπωση | Αυξήστε το `Resolution` (π.χ., 300) ή προσαρμόστε το `Scale`. |
| **Ασυμφωνία χρώματος** | Το φόντο εμφανίζεται διαφανές | Ορίστε το `options.BackgroundColor` σε ένα στερεό χρώμα. |
