# Πώς να ορίσετε άδεια για το Aspose.TeX (C#)

## Εισαγωγή

Αν σχεδιάζετε να δουλέψετε με αρχεία TeX σε μια εφαρμογή C#, το πρώτο πράγμα που πρέπει να κάνετε είναι **πώς να ορίσετε άδεια** για το Aspose.TeX. Η ρύθμιση μιας μετρημένης άδειας δεν αφαιρεί μόνο τους περιορισμούς χρόνου εκτέλεσης, αλλά σας δίνει επίσης πρόσβαση στο πλήρες σύνολο λειτουργιών της βιβλιοθήκης. Σε αυτόν τον οδηγό θα περάσουμε από όλη τη διαδικασία — από τη λήψη του SDK μέχρι την επαλήθευση ότι η άδεια είναι ενεργή — ώστε να μπορείτε να αρχίσετε να δημιουργείτε λύσεις με βάση το TeX αμέσως.

## Γρήγορες Απαντήσεις
- **Τι είναι μια μετρημένη άδεια;** Ένα ελαφρύ μοντέλο άδειας που επικυρώνει τη χρήση μέσω δημόσιων/ιδιωτικών κλειδιών χωρίς την ανάγκη τοπικού αρχείου άδειας.  
- **Χρειάζομαι άδεια για ανάπτυξη;** Ναι, απαιτείται μια μετρημένη άδεια τόσο για ανάπτυξη όσο και για παραγωγή ώστε να ξεκλειδωθούν όλες οι λειτουργίες.  
- **Ποιες εκδόσεις .NET υποστηρίζονται;** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7.  
- **Μπορώ να αλλάξω τα κλειδιά αργότερα;** Απόλυτα — απλώς καλέστε ξανά το `SetMeteredKey` με τα νέα κλειδιά.  
- **Πώς επιβεβαιώνω ότι η άδεια λειτουργεί;** Χρησιμοποιήστε το `Aspose.TeX.Metered.IsMetered()` για να λάβετε αποτέλεσμα true/false.

## Τι είναι μια Μετρημένη Άδεια;

Μια μετρημένη άδεια για το Aspose.TeX λειτουργεί στέλνοντας ένα ελαφρύ αίτημα στον διακομιστή αδειών της Aspose κάθε φορά που η εφαρμογή σας ξεκινά. Ο διακομιστής επικυρώνει τα δημόσια και ιδιωτικά κλειδιά που παρέχετε και επιστρέφει ένα διακριτικό χρήσης. Αυτή η προσέγγιση εξαλείφει την ανάγκη αποστολής φυσικού αρχείου άδειας και καθιστά εύκολη την εναλλαγή κλειδιών εάν χρειαστεί.

## Γιατί να χρησιμοποιήσετε μια Μετρημένη Άδεια για το Aspose.TeX;

- **Χωρίς ανάπτυξη αρχείου** – Τα κλειδιά ενσωματώνονται απευθείας στον κώδικα.  
- **Εύκολη εναλλαγή κλειδιών** – Ενημερώστε τα κλειδιά χωρίς επανεγκατάσταση αρχείου άδειας.  
- **Ακριβής παρακολούθηση χρήσης** – Η Aspose καταγράφει κάθε ενεργοποίηση, βοηθώντας σας να παραμείνετε εντός των ορίων της συνδρομής σας.  
- **Πλήρης πρόσβαση σε λειτουργίες** – Όλες οι δυνατότητες του API ξεκλειδώνουν μόλις η άδεια επικυρωθεί.

2. **Κλειδιά Μετρημένης Άδειας** – Αποκτήστε τα δημόσια και ιδιωτικά κλειδιά μετρημένης άδειας από τον λογαριασμό σας στην Aspose. Αν δεν έχετε λογαριασμό, μπορείτε να εγγραφείτε [εδώ](https://purchase.aspose.com/buy).  
3. **Περιβάλλον Ανάπτυξης C#** – Visual Studio (οποιαδήποτε πρόσφατη έκδοση) ή άλλο IDE C# της επιλογής σας.  

> **Συμβουλή:** Διατηρήστε τα κλειδιά μετρημένης άδειας σε ασφαλή αποθήκη ρυθμίσεων (π.χ., Azure Key Vault) αντί να τα κωδικοποιείτε σκληρά στον κώδικα.

## Εισαγωγή Namespaces

In your C# project, start by importing the Aspose.TeX namespace:

```csharp
using Aspose.TeX;
```

## Οδηγός Βήμα‑βήμα για τη Ρύθμιση Άδειας Aspose σε C#

### Βήμα 1: Ορισμός Μετρημένης Άδειας (πώς να ορίσετε άδεια)

Το πρώτο απόσπασμα κώδικα δείχνει ακριβώς **πώς να ορίσετε άδεια** χρησιμοποιώντας τα μετρημένα κλειδιά. Τοποθετήστε το νωρίς στη διαδικασία εκκίνησης της εφαρμογής σας (π.χ., `Main` ή `Startup.cs`).

```csharp
// ExStart:SetMeteredLicense
// Set metered public and private keys.
new Aspose.TeX.Metered().SetMeteredKey(
    "<type public key here>",
    "<type private key here>");
// ExEnd:SetMeteredLicense
```

Αντικαταστήστε το `<type public key here>` και το `<type private key here>` με τα κλειδιά που λάβατε από την Aspose.

### Βήμα 2: Ενσωμάτωση στο Πρότζεκτ σας

Αφού οριστεί η άδεια, μπορείτε ελεύθερα να χρησιμοποιήσετε οποιεσδήποτε κλάσεις Aspose.TeX — μεταγλώττιση LaTeX, μετατροπή σε PDF, εξαγωγή εικόνων κ.λπ. Δεν απαιτείται επιπλέον κώδικας αδειοδότησης.

### Βήμα 3: Επαλήθευση της Άδειας

Είναι καλή πρακτική να επιβεβαιώσετε ότι η άδεια εφαρμόστηκε επιτυχώς. Το παρακάτω απόσπασμα εκτυπώνει ένα σαφές μήνυμα στην κονσόλα.

```csharp
// ExStart:VerifyMeteredLicense
if (Aspose.TeX.Metered.IsMetered())
{
    Console.WriteLine("Metered license is set successfully!");
}
else
{
    Console.WriteLine("Metered license is not set!");
}
// ExEnd:VerifyMeteredLicense
```

Αν δείτε το «Metered license is set successfully!» είστε έτοιμοι να ξεκινήσετε.

## Συχνά Προβλήματα & Επίλυση

| Symptom | Likely Cause | Solution |
|---------|--------------|----------|
| `IsMetered()` returns **false** | Λάθος κλειδιά ή πρόβλημα συνδεσιμότητας δικτύου | Ελέγξτε ξανά τα κλειδιά, βεβαιωθείτε ότι το μηχάνημα μπορεί να φτάσει στο `license.aspose.com`. |
| Application throws **LicenseException** | Η άδεια ορίστηκε μετά τη χρήση των API του Aspose.TeX | Μετακινήστε τον κώδικα ορισμού άδειας στην αρχή του προγράμματος (πριν δημιουργηθούν οποιαδήποτε αντικείμενα Aspose.TeX). |
| Keys exposed in source control | Κίνδυνος ασφαλείας | Αποθηκεύστε τα κλειδιά σε μεταβλητές περιβάλλοντος ή σε ασφαλή θησαυροφυλάκιο και διαβάστε τα κατά την εκτέλεση. |
