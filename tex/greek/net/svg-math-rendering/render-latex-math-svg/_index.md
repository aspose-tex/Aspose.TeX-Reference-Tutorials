## Εισαγωγή Χώρων Ονομάτων

Το `Aspose.TeX` namespace παρέχει όλες τις κλάσεις που χρειάζεστε για την απόδοση εξισώσεων. Εισάγετέ το στην κορυφή του αρχείου σας:

```csharp
using Aspose.TeX;
```

Τώρα ας περάσουμε βήμα-βήμα τη διαδικασία απόδοσης.

## Βήμα 1: Δημιουργία Επιλογών Απόδοσης

MathRendererOptions είναι η βασική κλάση που περιέχει ρυθμίσεις για την απόδοση LaTeX σε διάφορες μορφές. Η SvgMathRendererOptions κληρονομεί από αυτήν και προσθέτει ιδιότητες ειδικές για SVG.

```csharp
using Aspose.TeX.Features;
```

## Βήμα 2: Καθορισμός Προεγγραφής

Η ιδιότητα Preamble σας επιτρέπει να προσθέσετε πακέτα και εντολές LaTeX που επεξεργάζονται πριν από την κύρια εξίσωση.

```csharp
// Create rendering options.
MathRendererOptions options = new SvgMathRendererOptions();
```

## Βήμα 3: Ορισμός Συντελεστή Κλιμάκωσης και Χρωμάτων

options.Scale ελέγχει το μέγεθος του παραγόμενου SVG, ενώ options.TextColor και options.BackgroundColor ορίζουν τα χρώματα προσκηνίου και φόντου.

```csharp
// Specify the preamble.
options.Preamble = @"\usepackage{amsmath}
\usepackage{amsfonts}
\usepackage{amssymb}
\usepackage{color}";
```

## Βήμα 4: Διαμόρφωση Επιλογών Εξόδου

OutputFile καθορίζει τη διαδρομή όπου θα αποθηκευτεί το παραγόμενο SVG, και options.EmbedFonts καθορίζει αν τα γραμματοσειρά ενσωματώνονται στο SVG.

```csharp
// Specify the scaling factor (e.g., 300%).
options.Scale = 3000;

// Specify the foreground color.
options.TextColor = System.Drawing.Color.Black;

// Specify the background color.
options.BackgroundColor = System.Drawing.Color.White;
```

## Βήμα 5: Απόδοση της Μαθηματικής Εξίσωσης LaTeX

MathRenderer είναι η μηχανή που παίρνει τη συμβολοσειρά LaTeX και τις επιλογές απόδοσης για να παραγάγει το τελικό έγγραφο SVG.

```csharp
// Specify the output stream for the log file.
options.LogStream = new System.IO.MemoryStream();

// Specify whether to show the terminal output on the console or not.
options.ShowTerminal = true;
```

## Βήμα 6: Εμφάνιση Αποτελεσμάτων

SvgDocument αντιπροσωπεύει το παραγόμενο SVG και μπορεί να αποθηκευτεί στο δίσκο ή να ροηθεί απευθείας σε απάντηση web.

```csharp
// Create the output stream for the formula image.
using (System.IO.Stream stream = System.IO.File.Open(
    System.IO.Path.Combine("Your Output Directory", "math-formula.svg"), System.IO.FileMode.Create))
{
    // Run rendering.
    new SvgMathRenderer().Render(@"\begin{equation*}
    e^x = x^{\color{red}0} + x^{\color{red}1} + \frac{x^{\color{red}2}}{2} + \frac{x^{\color{red}3}}{6} + \cdots = \sum_{n\geq 0} \frac{x^{\color{red}n}}{n!}
\end{equation*}", stream, options, out size);
}
```
