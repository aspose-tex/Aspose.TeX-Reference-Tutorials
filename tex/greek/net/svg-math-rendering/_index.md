## Συχνές Ερωτήσεις

**Q: Μπορώ να χρησιμοποιήσω τα παραγόμενα αρχεία SVG στο web χωρίς πρόσθετη μετατροπή;**  
A: Ναι—το SVG υποστηρίζεται εγγενώς από όλα τα σύγχρονα προγράμματα περιήγησης, οπότε μπορείτε να ενσωματώσετε το αποτέλεσμα απευθείας σε HTML ή CSS.

**Q: Πώς αλλάζω την προεπιλεγμένη γραμματοσειρά για τα αποδοθέντα μαθηματικά;**  
A: Χρησιμοποιήστε την ιδιότητα `FontFamily` της διαμόρφωσης `TeXRenderer` για να ορίσετε οποιαδήποτε εγκατεστημένη γραμματοσειρά TrueType/OpenType.

**Q: Είναι δυνατόν να αποδοθούν εξισώσεις LaTeX που περιλαμβάνουν χρώμα ή προσαρμοσμένα μακροεντολές;**  
A: Απολύτως. Το Aspose.TeX επεξεργάζεται τα τυπικά πακέτα χρώματος του LaTeX και σας επιτρέπει να ορίσετε μακροεντολές μέσω της μεθόδου `AddMacro`.

**Q: Ποιο μέγεθος θα έχει το παραγόμενο SVG;**  
A: Οι διαστάσεις του SVG υπολογίζονται αυτόματα βάσει του περιγράμματος της εξίσωσης, αλλά μπορείτε να τις παρακάμψετε χρησιμοποιώντας τις ρυθμίσεις `Width` και `Height`.

**Q: Υποστηρίζει η βιβλιοθήκη επεξεργασία παρτίδας πολλαπλών εξισώσεων;**  
A: Ναι—μπορείτε να κάνετε βρόχο σε μια συλλογή αλφαριθμητικών LaTeX και να αποδώσετε το καθένα σε δικό του αρχείο SVG με ελάχιστο κόστος.
