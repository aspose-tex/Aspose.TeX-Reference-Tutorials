## Πώς να Μετατρέψετε LaTeX σε PNG σε Java – Οδηγός Βήμα‑Βήμα

### Βήμα 1: Προσθέστε το Aspose.TeX στο Έργο σας
Προσθέστε το Maven coordinate (ή κατεβάστε το JAR) και ενσωματώστε το στο `pom.xml` ή `build.gradle`. Αυτό σας δίνει πρόσβαση στο πακέτο `com.aspose.tex`.

### Βήμα 2: Προετοιμάστε την Πηγή LaTeX
Μπορείτε να φορτώσετε LaTeX από συμβολοσειρά, αρχείο ή οποιοδήποτε `InputStream`. Διατηρήστε την πηγή κωδικοποιημένη σε UTF‑8 για να διατηρηθούν οι ειδικοί χαρακτήρες.

### Βήμα 3: Διαμορφώστε τις Επιλογές Εξαγωγής PNG
Δημιουργήστε μια παρουσία της `PngExportOptions` (ή της αντίστοιχης κλάσης) και ορίστε ιδιότητες όπως `resolutionDpi`, `imageWidth`, `imageHeight` και `backgroundColor`. Αυτές οι ρυθμίσεις επηρεάζουν άμεσα την οπτική ποιότητα και το μέγεθος του παραγόμενου PNG.

### Βήμα 4: Εκτελέστε τη Μετατροπή
Δημιουργήστε την κλάση `Document` με την πηγή LaTeX, στη συνέχεια καλέστε τη μέθοδο `save`, περνώντας τις επιλογές εξαγωγής και τη διαδρομή του αρχείου προορισμού (ή ένα `OutputStream`). Η βιβλιοθήκη διαχειρίζεται την ανάλυση, τη στοιχειοθέτηση και τη ραστεροποίηση εσωτερικά.

### Βήμα 5: Επαληθεύστε το Αποτέλεσμα
Ανοίξτε το παραγόμενο PNG σε οποιονδήποτε προβολέα εικόνας ή ενσωματώστε το σε μια ιστοσελίδα για να βεβαιωθείτε ότι η απόδοση ταιριάζει με τις προσδοκίες. Προσαρμόστε τις επιλογές εξαγωγής αν χρειάζεστε υψηλότερη ανάλυση ή διαφορετικό χρώμα φόντου.

> **Συμβουλή επαγγελματία:** Για επεξεργασία παρτίδας, επαναχρησιμοποιήστε την ίδια παρουσία `PngExportOptions` σε πολλαπλές μετατροπές ώστε να μειώσετε το κόστος δημιουργίας αντικειμένων.

## Συνηθισμένα Προβλήματα & Επίλυση

| Συμπτωμα | Πιθανή Αιτία | Διόρθωση |
|----------|--------------|----------|
| Blank PNG output | Missing font files in the classpath | Add the required LaTeX fonts or use the built‑in font fallback. |
| Low‑resolution image | Default DPI (72) is too low for your use case | Increase `resolutionDpi` in `PngExportOptions`. |
| Out‑of‑memory error on large documents | Whole document is loaded into memory | Process the LaTeX in chunks or increase JVM heap size (`-Xmx`). |
