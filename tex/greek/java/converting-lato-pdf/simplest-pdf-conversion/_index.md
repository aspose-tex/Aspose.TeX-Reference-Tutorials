## Οδηγός Βήμα‑Βήμα

### Βήμα 1: Ρύθμιση Επιλογών Μετατροπής

```java
TeXOptions options = TeXOptions.consoleAppOptions(TeXConfig.objectLaTeX());
```

Εδώ λέμε στο Aspose.TeX να αντιμετωπίζει την είσοδο ως *Object LaTeX*, που είναι η πιο κοινή μορφή για αυτόνομα έγγραφα.

### Βήμα 2: Ορισμός Καταλόγου Εργασίας Εξόδου

```java
options.setOutputWorkingDirectory(new OutputFileSystemDirectory("Your Output Directory"));
```

Αντικαταστήστε το `"Your Output Directory"` με την απόλυτη ή σχετική διαδρομή όπου θέλετε να αποθηκευτεί το παραγόμενο PDF.

### Βήμα 3: Αρχικοποίηση PDF Save Options

```java
options.setSaveOptions(new PdfSaveOptions());
```

`PdfSaveOptions` σας δίνει έλεγχο πάνω στις ρυθμίσεις ειδικές για PDF (συμπίεση, ενσωμάτωση γραμματοσειρών κλπ.). Η προεπιλεγμένη διαμόρφωση λειτουργεί για τις περισσότερες περιπτώσεις.

### Βήμα 4: Εκτέλεση Μετατροπής LaTeX σε PDF

```java
new TeXJob("Your Input Directory" + "hello-world.ltx", new PdfDevice(), options).run();
```

Δώστε την πλήρη διαδρομή στο αρχείο πηγής `.ltx`. Το `PdfDevice` λέει στο Aspose.TeX να αποδώσει την έξοδο ως έγγραφο PDF.

> **Pro tip:** Αν χρειάζεται να μετατρέψετε πολλαπλά αρχεία, κάντε βρόχο στον κατάλογο και καλέστε `new TeXJob(...).run();` για κάθε αρχείο.

Συγχαρητήρια! Εκτελέσατε μια **java latex σε pdf μετατροπή** με μόλις λίγες δηλώσεις.
