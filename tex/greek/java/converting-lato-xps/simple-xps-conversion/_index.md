## Εισαγωγή Πακέτων

Το μπλοκ `import` φέρνει τις απαραίτητες κλάσεις στο πεδίο ορατότητας. Η διατήρηση αυτής της ακριβούς λίστας εξασφαλίζει ότι ο κώδικας θα μεταγλωττιστεί χωρίς τροποποίηση.

```java
package com.aspose.tex.LaTeXXpsConversionSimplest;

import java.io.ByteArrayInputStream;
import java.io.IOException;
import java.util.Calendar;
import java.util.GregorianCalendar;

import com.aspose.tex.InputFileSystemDirectory;

import com.aspose.tex.TeXOptions;
import com.aspose.tex.rendering.XpsDevice;
import com.aspose.tex.rendering.XpsSaveOptions;

import util.Utils;
```

Τώρα ας περάσουμε από τα βήματα μετατροπής, εξηγώντας το σκοπό κάθε τμήματος κώδικα.

## Βήμα 1: Ορισμός Καταλόγων Εισόδου και Εξόδου

Πρέπει να ενημερώσετε το Aspose.TeX πού να βρει το αρχείο πηγής `.ltx` και πού να γράψει το παραγόμενο αρχείο XPS.

```java
options.setInputWorkingDirectory(new InputFileSystemDirectory("Your Input Directory"));
options.setOutputWorkingDirectory(new OutputFileSystemDirectory("Your Output Directory"));
```

*Tip:* Αντικαταστήστε το `"Your Input Directory"` και το `"Your Output Directory"` με απόλυτες ή σχετικές διαδρομές που υπάρχουν στο σύστημά σας.

## Βήμα 2: Ρύθμιση Επιλογών TeX

Αυτές οι επιλογές ελέγχουν τη συμπεριφορά της μηχανής LaTeX κατά τη μετατροπή. Προσαρμόστε τις ώστε να ταιριάζουν με τις απαιτήσεις του εγγράφου σας.  
Το TeXOptions είναι η κλάση που περιέχει αυτές τις ρυθμίσεις διαμόρφωσης.

```java
options.setInteraction(Interaction.NonstopMode);
options.setDateTime(new GregorianCalendar(2022, Calendar.DECEMBER, 18).getTime());
options.ignoreMissingPackages(true);
options.noLigatures(true);
options.repeat(true);
```

- **Interaction mode** – `NonstopMode` συνεχίζει την επεξεργασία ακόμη και αν συναντήσει σφάλματα.  
- **DateTime** – Ορίζει την ημερομηνία που εμφανίζεται στη σελίδα τίτλου του εγγράφου.  
- **ignoreMissingPackages** – Αποτρέπει την αποτυχία της εργασίας όταν δεν βρεθεί κάποιο πακέτο.  
- **noLigatures** – Απενεργοποιεί τις τυπογραφικές συνδέσεις αν προτιμάτε απλούς χαρακτήρες.  
- **repeat** – Επιτρέπει στη μηχανή να επανεκτελεστεί για διασταυρούμενες αναφορές.

## Βήμα 3: Αρχικοποίηση Επιλογών Αποθήκευσης XPS

Το XpsSaveOptions διαμορφώνει τις παραμέτρους εξόδου XPS όπως η συμπίεση και το μέγεθος σελίδας.

```java
options.setSaveOptions(new XpsSaveOptions());
```

## Βήμα 4: Προσαρμογή Επιλογών Αποθήκευσης XPS

Ρυθμίστε λεπτομερώς πώς η έξοδος XPS διαχειρίζεται τύπους, γραφικά και γραμματοσειρές.  
`rasterizeFormulas` καθορίζει αν οι τύποι αποδίδονται ως εικόνες.

```java
options.getSaveOptions().rasterizeFormulas(true);
options.getSaveOptions().rasterizeIncludedGraphics(true);
options.getSaveOptions().subsetFonts(true);
```

- **rasterizeFormulas** – Μετατρέπει τις μαθηματικές εκφράσεις σε εικόνες, εξασφαλίζοντας ότι εμφανίζονται σωστά σε οποιονδήποτε προβολέα XPS.  
- **rasterizeIncludedGraphics** – Αναγκάζει τα ενσωματωμένα γραφικά να ραστεροποιηθούν, κάτι που μπορεί να βελτιώσει τη συμβατότητα.  
- **subsetFonts** – Ενσωματώνει μόνο τα γλύφους που χρησιμοποιούνται στο έγγραφο, μειώνοντας το μέγεθος του αρχείου έως και 40 % για τυπικά τιμολόγια.

## Βήμα 5: Εκτέλεση Μετατροπής LaTeX σε XPS

Το XpsDevice γράφει το αποδοθέν περιεχόμενο σε αρχείο XPS.  
`TeXJob` συνδέει το αρχείο εισόδου, τη συσκευή εξόδου (`XpsDevice`) και όλες τις επιλογές που διαμορφώσατε.

```java
new TeXJob("Your Input Directory" + "sample.ltx", new XpsDevice(), options).run();
```

Μετά το τέλος της κλήσης `run()`, θα βρείτε το `sample.xps` στον κατάλογο εξόδου που καθορίσατε νωρίτερα. Αυτό το αρχείο XPS μπορεί να σταλεί απευθείας σε εκτυπωτή ή να ενσωματωθεί σε ένα πακέτο τιμολογίων PDF.

## Πρόσθετα Παραδείγματα

### Χρήση InputStream (Μετατροπή συμβολοσειράς LaTeX άμεσα)

Αν προτιμάτε να τροφοδοτήσετε την πηγή LaTeX από τη μνήμη αντί από αρχείο, τυλίξτε την πηγή σε ένα `ByteArrayInputStream`.

```java
new TeXJob(new ByteArrayInputStream(
    "\\documentclass{article} \\begin{document} Hello, World! \\end{document}".getBytes("ASCII")),
    new XpsDevice(), options).run();
```

### Χρήση Main Input Terminal (Αφήστε το Aspose.TeX να εντοπίσει το αρχείο αυτόματα)

Όταν δεν χρειάζεται να καθορίσετε ρητά τη διαδρομή του αρχείου, μπορείτε να βασιστείτε στο προεπιλεγμένο input terminal.

```java
new TeXJob(new XpsDevice(), options).run();
```
