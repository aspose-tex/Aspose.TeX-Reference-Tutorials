## Veelgestelde vragen

**Q: Kan ik Aspose.TeX gebruiken om LaTeX naar SVG te converteren in een webapplicatie?**  
A: Ja. De bibliotheek werkt op elke Java‑runtime, waardoor hij geschikt is voor server‑side rendering in webapps.

**Q: Hoe kan ik de terminaloutput vastleggen bij het converteren van LaTeX naar SVG?**  
A: Gebruik de *override job name*‑ en *write terminal output*‑opties; u kunt de output naar een bestand of een ZIP‑archief sturen zoals getoond in de gerelateerde tutorials.

**Q: Is het mogelijk om zowel figuren als wiskunde naar SVG te renderen in één enkele run?**  
A: Absoluut. U kunt de renderer configureren om meerdere LaTeX‑fragmenten te verwerken, waarbij elk een eigen SVG‑bestand produceert.

**Q: Heb ik een speciale licentie nodig voor SVG‑output?**  
A: Een standaard Aspose.TeX‑licentie dekt alle renderformaten, inclusief SVG.

**Q: Welke Java‑versie is vereist?**  
A: Aspose.TeX ondersteunt Java 8 en latere versies.
