## Veelgestelde vragen

**Q: Kan ik de gegenereerde SVG‑bestanden op het web gebruiken zonder extra conversie?**  
A: Ja—SVG wordt natively ondersteund door alle moderne browsers, dus je kunt de output direct in HTML of CSS insluiten.

**Q: Hoe wijzig ik het standaardlettertype voor de gerenderde wiskunde?**  
A: Gebruik de `FontFamily`‑eigenschap van de `TeXRenderer`‑configuratie om elk geïnstalleerd TrueType/OpenType‑lettertype op te geven.

**Q: Is het mogelijk om LaTeX‑vergelijkingen te renderen die kleur of aangepaste macro’s bevatten?**  
A: Absoluut. Aspose.TeX verwerkt standaard LaTeX‑kleurpakketten en stelt je in staat macro’s te definiëren via de `AddMacro`‑methode.

**Q: Hoe groot wordt de gegenereerde SVG?**  
A: De SVG‑afmetingen worden automatisch berekend op basis van de omhullende box van de vergelijking, maar je kunt ze overschrijven met de instellingen `Width` en `Height`.

**Q: Ondersteunt de bibliotheek batch‑verwerking van meerdere vergelijkingen?**  
A: Ja—je kunt door een collectie LaTeX‑strings itereren en elke string naar een eigen SVG‑bestand renderen met minimale overhead.
