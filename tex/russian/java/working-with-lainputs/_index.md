## Часто задаваемые вопросы

**Q: Могу ли я использовать Aspose.TeX для генерации PDF из LaTeX в веб‑приложении?**  
A: Да. Библиотека работает в любой Java‑среде, включая сервлет‑контейнеры и приложения Spring Boot.

**Q: Нужно ли устанавливать дистрибуцию TeX на сервере?**  
A: Нет. Aspose.TeX включает собственный TeX‑движок, поэтому внешних зависимостей нет.

**Q: Как обрабатывать пользовательские пакеты LaTeX, которые не включены в комплект?**  
A: Вы можете добавить файлы пакетов в тот же каталог, что и ваш источник, или указать пользовательский путь к пакетам через `CompilationOptions`.

**Q: Можно ли напрямую передавать сгенерированный PDF клиенту без сохранения на диск?**  
A: Конечно. Используйте метод `save(OutputStream)`, чтобы записать PDF в поток HTTP‑ответа.

**Q: Какие версии Java поддерживаются?**  
A: Aspose.TeX поддерживает Java 8 и новее, включая Java 11, 17 и более новые LTS‑выпуски.
