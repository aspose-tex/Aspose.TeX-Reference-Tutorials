Генерация PDF из LaTeX означает преобразование высококачественного, основанного на разметке LaTeX‑исходника в конечный PDF‑документ. Aspose.TeX обрабатывает компиляцию внутри, поэтому вам не требуется внешнее TeX‑распределение.

- **Вывод с высокой точностью** – полученный PDF сохраняет точный макет и типографическое качество оригинального LaTeX‑исходника.

## Часто задаваемые вопросы

**В:** Могу ли я использовать Aspose.TeX для генерации PDF из LaTeX в веб‑приложении?  
**О:** Да. Библиотека работает в любой Java‑среде, включая сервлет‑контейнеры и приложения Spring Boot.

**В:** Нужно ли устанавливать TeX‑распределение на сервер?  
**О:** Нет. Aspose.TeX включает собственный TeX‑движок, поэтому внешних зависимостей нет.

**В:** Как обрабатывать пользовательские LaTeX‑пакеты, которые не включены в комплект?  
**О:** Вы можете добавить файлы пакетов в тот же каталог, что и ваш источник, или указать пользовательский путь к пакетам через `CompilationOptions`.

**В:** Можно ли передавать сгенерированный PDF напрямую клиенту без сохранения на диск?  
**О:** Конечно. Используйте метод `save(OutputStream)`, чтобы записать PDF в поток HTTP‑ответа.

**В:** Какие версии Java поддерживаются?  
**О:** Aspose.TeX поддерживает Java 8 и более новые версии, включая Java 11, 17 и более новые LTS‑выпуски.
