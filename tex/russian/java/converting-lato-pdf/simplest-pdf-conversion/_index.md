## Импорт пакетов

Сначала импортируйте классы, необходимые для преобразования. Ниже приведённый фрагмент кода не изменён по сравнению с оригиналом, что гарантирует его работоспособность.

```java
package com.aspose.tex.LaTeXPdfConversionSimplest;

import java.io.IOException;

import com.aspose.tex.OutputFileSystemDirectory;
import com.aspose.tex.TeXConfig;
import com.aspose.tex.TeXJob;
```

## Пошаговое руководство

### Шаг 1: Настройка параметров преобразования

```java
TeXOptions options = TeXOptions.consoleAppOptions(TeXConfig.objectLaTeX());
```

Здесь мы указываем Aspose.TeX рассматривать ввод как *Object LaTeX*, что является наиболее распространённым форматом для автономных документов.

### Шаг 2: Установка рабочей директории вывода

```java
options.setOutputWorkingDirectory(new OutputFileSystemDirectory("Your Output Directory"));
```

Замените `"Your Output Directory"` на абсолютный или относительный путь, где вы хотите сохранить полученный PDF.

### Шаг 3: Инициализация параметров сохранения PDF

```java
options.setSaveOptions(new PdfSaveOptions());
```

`PdfSaveOptions` предоставляет контроль над настройками PDF (сжатие, встраивание шрифтов и т.д.). Конфигурация по умолчанию подходит для большинства сценариев.

### Шаг 4: Запуск преобразования LaTeX в PDF

```java
new TeXJob("Your Input Directory" + "hello-world.ltx", new PdfDevice(), options).run();
```

Укажите полный путь к вашему исходному файлу `.ltx`. `PdfDevice` сообщает Aspose.TeX отрисовать вывод в виде PDF‑документа.

> **Совет:** Если вам нужно конвертировать несколько файлов, пройдитесь по каталогу в цикле и вызовите `new TeXJob(...).run();` для каждого файла.

Поздравляем! Вы выполнили **java latex to pdf conversion** всего лишь несколькими инструкциями.
