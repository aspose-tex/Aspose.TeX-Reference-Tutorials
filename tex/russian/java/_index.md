## Как конвертировать LaTeX в PDF на Java?

Класс `TeXDocument` представляет файл исходного кода LaTeX и предоставляет методы для загрузки, обработки и рендеринга документа.  
Метод `save` записывает обработанный документ в выбранный формат вывода, например PDF.

Загрузите ваш файл `.tex` с помощью класса `TeXDocument` и вызовите `save("output.pdf", SaveFormat.Pdf)` — эта двухстрочная последовательность выполняет полную **как конвертировать latex** операцию. Библиотека автоматически разрешает пакеты, шрифты и изображения, создавая PDF, соответствующий оригинальному макету LaTeX. Для больших документов включите режим потоковой передачи, чтобы потребление памяти оставалось ниже 100 МБ даже при файлах более 500 МБ.
