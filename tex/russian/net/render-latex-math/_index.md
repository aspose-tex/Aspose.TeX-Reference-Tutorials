## Часто задаваемые вопросы

**Q: Можно ли отрендерить несколько уравнений в одном PNG?**  
A: Да. Объедините строки LaTeX или используйте класс `Document` для размещения нескольких формул перед рендерингом.

**Q: Поддерживает ли Aspose.TeX другие форматы изображений?**  
A: Абсолютно. Помимо PNG, можно выводить JPEG, BMP, GIF и TIFF, изменив свойство `ImageFormat`.

**Q: Как ускорить рендеринг при пакетной обработке?**  
A: Переиспользуйте один экземпляр `Renderer` и предварительно загрузите необходимые шрифты; это уменьшит накладные расходы на каждое преобразование.

**Q: Можно ли получить прозрачный PNG из LaTeX?**  
A: Установите цвет фона `Color.Transparent` в параметрах рендеринга перед вызовом `Save`.

**Q: Какая версия .NET требуется?**  
A: Aspose.TeX поддерживает .NET Framework 4.5+, .NET Core 3.1+, .NET 5 и .NET 6.
