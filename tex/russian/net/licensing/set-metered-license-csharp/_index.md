# Как установить лицензию для Aspose.TeX (C#)

## Введение

Если вы планируете работать с файлами TeX в приложении на C#, первое, что вам нужно сделать, — **как установить лицензию** для Aspose.TeX. Установка лицензии с измерением (metered license) не только снимает ограничения во время выполнения, но и предоставляет доступ к полному набору функций библиотеки. В этом руководстве мы пройдем весь процесс — от загрузки SDK до проверки активности лицензии — чтобы вы могли сразу начать создавать решения на основе TeX.

## Быстрые ответы

- **Что такое лицензия с измерением?** Легковесная модель лицензирования, которая проверяет использование с помощью публичных/приватных ключей без необходимости локального лицензионного файла.  
- **Нужна ли лицензия для разработки?** Да, лицензия с измерением требуется как для разработки, так и для продакшена, чтобы разблокировать все функции.  
- **Какие версии .NET поддерживаются?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7.  
- **Можно ли позже изменить ключи?** Конечно — просто вызовите `SetMeteredKey` снова с новыми ключами.  
- **Как подтвердить, что лицензия работает?** Используйте `Aspose.TeX.Metered.IsMetered()`, чтобы получить результат true/false.

## Что такое лицензия с измерением?

Лицензия с измерением для Aspose.TeX работает, отправляя легковесный запрос на сервер лицензирования Aspose каждый раз при запуске вашего приложения. Сервер проверяет предоставленные вами публичный и приватный ключи и возвращает токен использования. Такой подход устраняет необходимость поставлять физический лицензионный файл и упрощает ротацию ключей при необходимости.

## Зачем использовать лицензию с измерением для Aspose.TeX?

- **Отсутствие развертывания файлов** – Ключи встраиваются непосредственно в код.  
- **Лёгкая ротация ключей** – Обновляйте ключи без повторного развертывания лицензионного файла.  
- **Точное отслеживание использования** – Aspose фиксирует каждую активацию, помогая вам оставаться в пределах лимитов подписки.  
- **Полный доступ к функциям** – Все возможности API разблокируются после проверки лицензии.

## Требования

Прежде чем начать, убедитесь, что у вас готовы следующие элементы:

1. **Aspose.TeX for .NET Library** – Скачайте и установите библиотеку со [страницы загрузки](https://releases.aspose.com/tex/net/).  
2. **Metered License Keys** – Получите публичный и приватный ключи с измерением из вашего аккаунта Aspose. Если у вас нет аккаунта, вы можете зарегистрироваться [здесь](https://purchase.aspose.com/buy).  
3. **C# Development Environment** – Visual Studio (любая современная версия) или другая IDE для C# по вашему выбору.  

> **Совет:** Храните свои ключи с измерением в безопасном хранилище конфигураций (например, Azure Key Vault), а не в виде жёстко закодированных значений.

## Установка лицензии Aspose в C# – Пошаговое руководство

### Шаг 1: Установить лицензию с измерением (how to set license)

Первый фрагмент кода показывает точно **как установить лицензию** с использованием ключей с измерением. Разместите его в начале процедуры запуска вашего приложения (например, `Main` или `Startup.cs`).

```csharp
// ExStart:SetMeteredLicense
// Set metered public and private keys.
new Aspose.TeX.Metered().SetMeteredKey(
    "<type public key here>",
    "<type private key here>");
// ExEnd:SetMeteredLicense
```

Замените `<type public key here>` и `<type private key here>` полученными от Aspose ключами.

### Шаг 2: Интегрировать в ваш проект

После установки лицензии вы можете свободно использовать любые классы Aspose.TeX — компиляцию LaTeX, конвертацию в PDF, извлечение изображений и т.д. Дополнительный код лицензирования не требуется.

### Шаг 3: Проверить лицензию

Хорошей практикой является подтверждение успешного применения лицензии. Следующий фрагмент выводит чёткое сообщение в консоль.

```csharp
// ExStart:VerifyMeteredLicense
if (Aspose.TeX.Metered.IsMetered())
{
    Console.WriteLine("Metered license is set successfully!");
}
else
{
    Console.WriteLine("Metered license is not set!");
}
// ExEnd:VerifyMeteredLicense
```

Если вы видите «Metered license is set successfully!», вы готовы к работе.

## Распространённые проблемы и их устранение

| Симптом | Вероятная причина | Решение |
|---------|-------------------|----------|
| `IsMetered()` возвращает **false** | Неправильные ключи или проблема с сетевым подключением | Проверьте ключи ещё раз, убедитесь, что машина может достичь `license.aspose.com`. |
| Приложение бросает **LicenseException** | Лицензия установлена после использования API Aspose.TeX | Переместите код установки лицензии в самое начало программы (до создания любых объектов Aspose.TeX). |
| Ключи раскрыты в системе контроля версий | Риск безопасности | Храните ключи в переменных окружения или безопасном хранилище и считывайте их во время выполнения. |
