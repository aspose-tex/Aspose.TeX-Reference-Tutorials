## Пошаговое руководство по рендерингу LaTeX в PNG

### Шаг 1: Настройка параметров рендеринга (FigureRendererOptions)

Создайте объект `FigureRendererOptions` и настройте разрешение, преамбулу, коэффициент масштабирования, цвет фона и параметры логирования. Регулировка **latex png resolution** напрямую влияет на чёткость конечного изображения.

```csharp
FigureRendererOptions options = new PngFigureRendererOptions() { Resolution = 150 };
options.Preamble = "\\usepackage{pict2e}";
options.Scale = 3000;
options.BackgroundColor = System.Drawing.Color.White;
options.LogStream = new System.IO.MemoryStream();
options.ShowTerminal = true;
```

### Шаг 2: Определение выходного потока и размеров

Подготовьте выходной поток, в который будет сохранён PNG, и структуру `SizeF` для получения размеров отрендеренного изображения. Здесь вы **создаёте PNG из LaTeX** на диске.

```csharp
System.Drawing.SizeF size = new System.Drawing.SizeF();
using (System.IO.Stream stream = System.IO.File.Open(
   System.IO.Path.Combine("Your Output Directory", "text-and-formula.png"), System.IO.FileMode.Create))
```

### Шаг 3: Запуск процесса рендеринга

Передайте исходный LaTeX, выходной поток, настроенные параметры и переменную размера рендереру. Рендерер преобразует окружение picture LaTeX в **высококачественный LaTeX PNG**.

```csharp
new PngFigureRenderer().Render(@"\setlength{\unitlength}{0.8cm}
\begin{picture}(6,5)
    % LaTeX figure code goes here
\end{picture}", stream, options, out size);
```

## Высококачественный LaTeX PNG: настройка разрешения и масштаба

Если требуется более чёткое изображение для печати, увеличьте `Resolution` (например, 300 dpi) или коэффициент `Scale`. Учтите, что большие значения повышают расход памяти, поэтому протестируйте настройки, подходящие для вашей среды развертывания.

## Распространённые проблемы и решения

| Проблема | Причина | Решение |
|----------|---------|----------|
| **Пустой PNG** | Выходной поток не сброшен или не закрыт | Убедитесь, что блок `using` завершён до чтения файла. |
| **Отсутствуют пакеты** | LaTeX‑код использует пакет, которого нет в преамбуле | Добавьте необходимый `\usepackage{...}` в `options.Preamble`. |
| **Низкое разрешение** | Стандартный DPI слишком низок для печати | Увеличьте `Resolution` (например, 300) или настройте `Scale`. |
| **Несоответствие цвета** | Фон выглядит прозрачным | Установите `options.BackgroundColor` в сплошной цвет. |
