## Часто задаваемые вопросы

**Q: Как я могу **convert latex to pdf** с помощью Aspose.TeX?**  
A: Создайте экземпляр `TeXDocument`, загрузите ваш `.tex`‑файл и вызовите `Save("output.pdf")`. Тот же API позволяет вызвать `Save("output.png")` или `Save("output.svg")` для других форматов.

**Q: Могу ли я **export latex as png** с пользовательским разрешением?**  
A: Да. Используйте объект `PngSaveOptions` для указания DPI, цвета фона и качества изображения перед сохранением.

**Q: Какой лучший способ **generate pdf from latex** в веб‑сервисе?**  
A: Разместите код конвертации в безсостоящем .NET Core API, при возможности кэшируйте полученный PDF и передавайте файл клиенту потоково.

**Q: Есть ли ограничение на размер исходного LaTeX, который я могу конвертировать?**  
A: Aspose.TeX обрабатывает большие документы, но для чрезвычайно крупных файлов рекомендуется увеличить лимит памяти или обрабатывать документ по частям.

**Q: Поддерживает ли Aspose.TeX **convert latex to svg** для векторной графики?**  
A: Абсолютно. Используйте `Save("output.svg")` или класс `SvgSaveOptions` для тонкой настройки вывода.
