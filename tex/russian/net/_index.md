- **What does Aspose.TeX do?** Он конвертирует исходные файлы TeX/LaTeX в PDF, PNG, SVG, XPS и другие форматы напрямую из .NET.

Генерация PDF из TeX означает взятие LaTeX‑исходного файла (или сырых команд TeX) и преобразование его в готовый PDF‑документ. Aspose.TeX для .NET обрабатывает весь конвейер компиляции внутри, поэтому вам не требуется внешняя LaTeX‑дистрибуция.

## Часто задаваемые вопросы

**Q: Как сгенерировать PDF из TeX без установки локального LaTeX‑дистрибутива?**  
A: Aspose.TeX встраивает TeX‑движок, поэтому вы можете вызвать API напрямую из вашего .NET‑кода и мгновенно получить PDF.

**Q: Как конвертировать LaTeX‑PDF в другие форматы, например SVG?**  
A: Используйте тот же API Aspose.TeX с соответствующим методом `Save` (например, `Save("output.svg", SaveOptions.CreateSvgSaveOptions())`). Это покрывается в руководствах «latex conversion to svg».

**Q: Можно ли рендерить уравнения LaTeX в PNG для веб‑миниатюр?**  
A: Да — руководство «render latex png» показывает, как вызвать `Save("eq.png", SaveOptions.CreatePngSaveOptions())` для получения PNG высокого разрешения.

**Q: Нужна ли отдельная лицензия для каждого формата вывода?**  
A: Нет. Одна лицензия Aspose.TeX покрывает все поддерживаемые форматы вывода (PDF, PNG, SVG, XPS и т.д.).

**Q: Какие типичные подводные камни при генерации PDF из TeX?**  
A: Отсутствие шрифтов, неподдерживаемые пакеты или неверные пути к файлам. Руководство «how to convert latex pdf» содержит советы по устранению этих проблем.
