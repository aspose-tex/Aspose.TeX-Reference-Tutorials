## 第 1 步：设置计量许可证

第一步涉及在 C# 代码中设置计量许可证。使用以下代码片段：

```csharp
// ExStart:设置计量许可证
//设置计量公钥和私钥。

    "<type public key here>",

```

代替`<type public key here>`和`<type private key here>`使用您实际计量的许可证密钥。

设置计量许可证后，将 Aspose.TeX 集成到您的项目中。您现在可以使用其功能，无需任何许可问题。
